# 표준 태그 분류 체계

> 이 위키에서 사용하는 모든 표준 태그의 정의와 사용 기준.
> 새 태그 추가 시 이 파일에 먼저 등록하고 사용합니다.

---

## 사용 원칙

- 태그는 **소문자+한글** 혼용 허용, 공백 없이 (예: `청소년금융`, `UX전략`)
- 하나의 파일에 **5개 이하** 권장 (핵심 태그만)
- 계층 태그 없음 (플랫 구조 유지)
- 약어는 프로젝트 전반에서 통일: `UX`, `NPS`, `BaaS`, `HNWI`

## 카테고리별 표준 태그

### 🏢 프로젝트 / 클라이언트

| 태그 | 정의 | 사용 위치 |
|------|------|---------|
| `아이부자` | 하나은행 아이부자 앱/서비스 직접 관련 | 전체 wiki (거의 모든 파일) |
| `하나은행` | 하나은행 기관 관련 | entities/, 전략 문서 |
| `컨설팅` | 이 컨설팅 프로젝트 운영 관련 | entities/, _meta/ |
| `이해관계자인터뷰` | 인터뷰 기반 자료 | sources/F04* |
| `리뉴얼` | 아이부자 리뉴얼 프로젝트 관련 | 전략 문서 |

### 👥 사용자 세그먼트

| 태그 | 정의 | 연령 기준 |
|------|------|---------|
| `알파세대` | 2010년 이후 출생 | 0~15세(2026년 기준) |
| `Z세대` | 1995~2010년 출생 | 15~31세 |
| `잘파세대` | Z세대+알파세대 통칭 | 0~31세 |
| `청소년` | 중·고등학생 중심 금융 사용자 | 13~18세 |

| `영유아` | 미취학 아동 | 0~5세 |
| `부모` | 자녀 금융 관리 주체로서의 부모 | 성인 |
| `HNWI` | 고자산 가정 (High Net Worth Individual) | — |

### 📚 도메인 / 주제

| 태그 | 정의 |
|------|------|
| `금융교육` | 금융 지식 전달 목적의 교육 활동 |
| `청소년금융` | 청소년 대상 금융 서비스·상품·행동 |

| `금융이해력` | Financial Literacy — 지식·태도·행동 종합 역량 |
| `금융소비자역량` | 디지털 환경 포함 금융 서비스 이용 역량 |
| `금융태도` | 금융에 대한 심리적 태도·가치관 |
| `재무관리행동` | 저축·지출·투자 등 실제 금융 행동 |
