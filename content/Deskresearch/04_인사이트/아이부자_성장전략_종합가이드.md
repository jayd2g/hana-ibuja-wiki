## 1. 아이부자의 성장 전략 개념도

```

시간

✨ 영유아 (0-5세)

         ↓ 자연스러운 진화

         ↓ 자연스러운 진화

│ 부모: 상담자 (선택적 개입)               │
│ 지표: 저축률 35-50%, 부모 상담 30-40%   │
└───────────────────────────────────────────┘
         ↓ 자연스러운 진화

│ STAGE 4: 고등학생 (16-18세)              │

│ 부모: 파트너 (최소 개입)                 │

└───────────────────────────────────────────┘

**핵심 구조 특징**:
1. **연속성**: 각 단계가 이전 단계를 기반으로 자연스럽게 진화
2. **확장성**: 서비스가 계속 확대되고, 기존 기능은 유지
3. **개인화**: 각 단계에서 "그 나이에 필요한 것"만 제공
4. **유연성**: 아이의 속도에 맞춰 다음 단계로 전환

## 2. Stage 1: 초등 저학년 (6-8세) - "거래 경험의 시작"

### 2.1 발달 특성 및 요구사항

#### 심리-인지 특성
```
❌ 할 수 없는 것:
  - 추상적 개념 이해
  - 미래 계획 (3개월 이상)
  - 가치 비교 판단
  - 금융 위험 인식

✅ 할 수 있는 것:
  - 직접적인 거래 경험
  - 즉시적인 보상/결과 인식
  - 부모의 지도 따르기
  - 반복 학습
```

#### 부모의 금융 역할
- **관찰**: 자녀의 거래 패턴 전체 모니터
- **지도**: 거의 모든 거래에 개입
- **교육**: "왜 저축해야 하나?"를 계속 설명
- **보상**: 목표 달성 시 칭찬과 격려

부모: 투명한 거래 모니터링 + 어떻게 지도할지 알려주기"

#### 주요 기능 상세

**1) 게임화된 용돈관리**
```
용돈 받기 → 사용하기/모으기 → 결과 확인

시각적 요소:
- 캐릭터 (아이가 선택)
- 레벨 시스템 (거래 수에 따라 레벨업)
- 뱃지 (저축 달성, 좋은 선택 등)
- 진행도 게이지
- 색상 + 사운드 피드백

게임 루프:
1. 용돈 받기 (💰 아이콘)
2. 거래하기 (🛍️ 클릭)
3. 저축하기 (🏦 드래그)
4. 결과 보기 (✨ 애니메이션)
5. 보상 얻기 (🎁 뱃지)
```

**2) 부모 상담 기능**
```
아이가 구매하고 싶을 때:
"엄마, 이거 사도 돼?" → 부모에게 알림

부모의 선택지:
- "좋은 생각이야!" (승인) → 거래 실행
- "기다려봐." (보류) → 대기 기간 기록
- "이것보다 저것이" (제안) → 대화 기회
- "한 번 생각해봐" (거절) → 학습 기회
```

**3) 초보 저축**
```
목표: 1-2개월 단위 단기 저축

예시:
┌─────────────────────────────────┐
│ 저축 목표: "여름방학 장난감"     │
│ 목표액: 10,000원                │
│ 기간: 5월 1일 ~ 6월 30일        │
├─────────────────────────────────┤
│ ████░░░░░░ 40% (4,000원)        │
│ 진행도: 한 달 진행              │
│ 예상 달성: 6월 30일             │
├─────────────────────────────────┤
│ "이대로 계속하면 목표 달성!"    │
└─────────────────────────────────┘

이 단계에서는 목표 달성 경험이 매우 중요
→ 작은 성공이 장기 동기 형성
```

**4) 부모 교육 콘텐츠**
```
매월 제공되는 부모 가이드:

【주제별 가이드】
1. "이 단계 아이의 금융 이해력은?"
   → 무엇을 기대하고, 무엇을 기대하면 안 되는가

2. "용돈을 주는 방법"
   → 금액, 주기, 조건 설정

3. "거래할 때 어떻게 말할까?"
   → 대화 스크립트 + 예시

4. "저축 목표를 함께 정하기"
   → 대화 방법 + 동기 부여

5. "잘못된 선택을 했을 때"
   → 비난하지 않고 배우게 하기
```

### 2.3 시스템 측정 지표 (Stage 1)

| 지표 | 초등 저학년 기준값 | 측정 방법 | 의미 |
|------|-----------------|---------|------|
| **월간 저축률** | 5% (0-15%) | 자동 계산 | 저축 개념 형성 |
| **용돈 소진 기간** | 3-5일 | 거래 타임스탬프 | 즉시 소비 특성 |
| **거래 빈도** | 25-35회/월 | 거래 수 | 충동적 구매 빈도 |
| **부모 상담 비율** | 80-100% | 상담 거래 수 | 부모 의존도 (정상) |
| **활성 목표 수** | 0-1개 | 목표 설정 | 단기 목표만 가능 |
| **목표 달성률** | 30-50% | 완수 목표 / 전체 | 충동성으로 인한 낮은 달성률 |

### 2.4 발달 심리학 기반 (왜 이 단계에서 이것을 할까?)

```
Piaget: 구체적 조작기 (7세경 진입)
└─ "직접 해봐야 이해한다"
   → 거래 경험이 필수

Erikson: 자율성 vs 수치심 (이 시기의 심리적 과제)
└─ "스스로 선택하고 해보고 싶다"
   → 용돈으로 "내가 사기"의 경험 필요

금융 사회화 이론: 부모의 영향이 가장 큼 (조선영, 2023)
└─ "부모와 함께 거래 경험하기"
   → 부모의 금전관 전수
```

### 2.5 법정/교육 기준 연계

```

## 3. Stage 2: 초등 고학년 (9-12세) - "계획 능력의 발현"

### 3.1 발달 특성 및 요구사항

#### 심리-인지 특성
```
변화:
✅ 추상적 사고 시작 (구체적 조작기 후반)
✅ 미래 계획 가능 (3-6개월)
✅ 인과관계 이해 (저축 → 목표 달성)
✅ 다른 사람 관점 이해 (또래 비교 시작)

새로운 과제:
❌ 또래 압박 (친구는 뭐 가지고 있어?)
❌ 자기 주장 (나는 이거 샀어)
❌ 계획의 중요성 (나중을 생각하기)
```

#### 부모의 금융 역할
- **관찰**: 자녀의 의사결정 과정 모니터
- **조언**: 모든 거래가 아닌 중요 거래에만 개입
- **협력**: "너 어떻게 생각해?"로 대화 확대
- **격려**: 계획적 선택에 대한 칭찬

### 6.2 부모의 역할 변화

```
Stage 1: "지도자"
└─ 거의 모든 거래에 개입
   "엄마, 이거 사도 돼?" → "좋은 선택이야"
   역할: 가르치기, 격려하기

         ↓ 자연스러운 진화

Stage 2: "조언자"
└─ 중요한 거래에만 개입
   "엄마, 이거 맞아?" → "너는 어떻게 생각해?"
   역할: 함께 고민하기, 선택 존중하기

         ↓ 자연스러운 진화

Stage 3: "상담자"
└─ 물어올 때만 개입
   "엄마, 신용카드가 뭐야?" → "깊이 있게 설명"
   역할: 조언하기, 판단 존중하기

         ↓ 자연스러운 진화

Stage 4: "파트너"
└─ 최소한의 개입
   "엄마, 이 주식 어때?" → "넌 어떻게 생각해?"
   역할: 지지하기, 필요시 조언하기
```
