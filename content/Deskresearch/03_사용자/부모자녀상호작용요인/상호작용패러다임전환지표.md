## 1. 개요 및 측정 프레임워크

### 1.1 V2.0의 변화

V1.0이 "부모의 주관적 관찰"에 기반했다면, V2.0은 **"아이부자 시스템에서 자동으로 추적되는 객관적 거래 데이터"**에 기반합니다. 이를 통해:

- **객관성**: 부모의 편견 없이 명확한 수치로 판단 가능
- **자동성**: 시스템이 자동으로 계산하고 추적
- **연속성**: 시간 경과에 따른 변화 추이 파악 가능
- **행동성**: 실제 금융 거래 행동을 기반으로 한 판단

### 1.2 측정 범주

| 범주 | 측정 항목 | 데이터 소스 |
|------|---------|----------|
| **거래 데이터** | 용돈액, 저축액, 저축률, 사용액 | 거래 내역 |
| **시간 지표** | 용돈 소진 기간, 저축 지속 기간, 사용 빈도 | 거래 타임스탬프 |
| **패턴 지표** | 구매 카테고리, 반복 구매, 부모 상담 비율 | 거래 분류 |
| **목표 지표** | 목표 설정 수, 달성률, 목표 기간 | 목표 데이터 |

## 2. 거래 데이터 기반 지표

### 2.1 월간 저축률 (Monthly Savings Rate)

**정의**: 월간 용돈 중에서 저축으로 돌린 금액의 비율
**공식**: (월간 저축액 / 월간 용돈액) × 100%
**아이부자 추적**: 자동 계산, 월별 대시보드 표시

**발달단계별 기준값**:

| 발달단계 | 평균 저축률 | 범위 | 해석 |
|---------|-----------|------|------|
| **초등 저학년 (6-8세)** | 5% | 0-15% | 저축 개념 형성 초기, 거의 충동적 소비 |
| **초등 고학년 (9-12세)** | 20% | 10-35% | 저축 습관 형성 중, 목표 의식 시작 |
| **중학생 (13-15세)** | 35% | 20-50% | 저축 의식 확립, 자발적 저축 시작 |
| **고등학생 (16-18세)** | 40% | 30-60% | 계획적 재무관리, 투자 고려 가능 |

**근거**: 조선영과 최현자(2023)의 연구에서 청소년의 저축 비율이 발달단계에 따라 비약적으로 증가한다는 것을 확인했습니다. 특히 초등 고학년에서 중학생으로의 전환점에서 저축률이 20%에서 35%로 증가합니다.

**아이부자 활용**:
- **<10% 구간**: "저축 습관 만들기" 미션 추천
- **10-20% 구간**: 목표 저축 기능 강화
- **20-35% 구간**: 목표 다층화 (단기/중기/장기)
- **>35% 구간**: 투자 상품 소개 시작

**사용자 피드백 예시**:
```
"지난 3개월 저축률: 22%
→ 초등 고학년 평균 수준입니다.
→ 다음 단계(중학생 수준 35%)에 도달하려면
   월 저축액을 5,000원 더 늘려보세요!"
```

### 2.2 용돈 소진 기간 (Days to Depletion)

**정의**: 용돈을 받은 후 완전히 소진될 때까지의 기간
**측정 방법**: 용돈 수령일부터 잔액 0이 될 때까지의 일수
**아이부자 추적**: 매월 자동 계산, 월별 추이 그래프

**발달단계별 기준값**:

| 발달단계       | 평균 소진 기간 | 범위      | 해석               |
| ---------- | -------- | ------- | ---------------- |
| **초등 저학년** | 3-5일     | 1-7일    | 즉시 소비 성향, 충동적 구매 |
| **초등 고학년** | 10-15일   | 7-20일   | 일부 계획적 사용 시작     |
| **중학생**    | 20-25일   | 15-30일  | 월말까지 계획적 사용      |
| **고등학생**   | 월 전체     | 25-30일+ | 저축 병행으로 월말까지 관리  |

**근거**: Shoda, Mischel과 Peake (1990)의 지연만족(Delayed Gratification) 연구에서 어린 시절의 충동 조절 능력이 평생의 금융 건강을 좌우한다고 보였습니다. 용돈 소진 기간은 이를 객관적으로 측정할 수 있는 지표입니다.

**중요 인사이트**:
- **감소 추이**: "5일→10일→20일"로 증가하는 추이 = 정상 발달
- **정체 또는 악화**: 같은 단계에 머물거나 더 빨리 소진 = 개입 필요
- **급격한 변화**: 갑자기 소진 기간이 크게 늘었다 = 목표 설정이나 부모 개입의 영향

**아이부자 활용**:
```
월간 소진 기간 추이:
- 1월: 4일 ▲
- 2월: 6일 ▲
- 3월: 10일 ▲  ← 개선 중!
- 4월: 12일 ▲

"계획적 사용이 점점 나아지고 있습니다.
 이대로 유지하면 2개월 후 중학생 수준에 도달할 수 있습니다!"
```

### 2.3 월간 사용 액수 대비 용돈 액수의 비율 (Usage to Allowance Ratio)

**정의**: 실제 사용액이 받은 용돈액의 몇 %인가
**공식**: (월간 사용액 / 월간 용돈액) × 100%
**참고**: 저축률 + 사용률 = 100%

**발달단계별 기준값**:

| 발달단계 | 사용률 | 저축률 | 해석 |
|---------|--------|--------|------|
| **초등 저학년** | 95% | 5% | 거의 모두 사용 |
| **초등 고학년** | 80% | 20% | 일부 저축 시작 |
| **중학생** | 65% | 35% | 저축과 사용의 균형 |
| **고등학생** | 60% | 40% | 저축 우선 의식 |

**아이부자 활용**:
```
"이번 달 재무 현황:
용돈 50,000원
├─ 저축: 10,000원 (20%) ○
├─ 사용: 40,000원 (80%) ●

목표: 저축 20% 유지 및 사용 80% 내에서
      카테고리별 관리 시작"
```

### 2.4 사용 금액의 카테고리 다양성 (Consumption Diversity)

**정의**: 사용 금액이 얼마나 다양한 카테고리에 분산되어 있는가
**측정**: 사용액이 집중된 카테고리의 비율

**예시**:
- **낮은 다양성**: 1개 카테고리(게임과금)에 50% 이상 집중
- **중간 다양성**: 3-4개 카테고리에 고르게 분산
- **높은 다양성**: 5개 이상 카테고리에 균형 있게 분산

**발달단계별 패턴**:

| 발달단계 | 전형적 패턴 | 위험 신호 |
|---------|-----------|---------|
| **초등 저학년** | 특정 카테고리 집중 (예: 장난감 70%) | 정상 |
| **초등 고학년** | 2-3개 카테고리 균형 | 1개 카테고리 >60% |
| **중학생** | 3-4개 카테고리 균형 | 1개 카테고리 >50% |
| **고등학생** | 4개 이상 카테고리 균형 | 1개 카테고리 >40% |

**아이부자 활용**:
```
"이번 달 사용 분석:
간식: 15,000원 (37.5%)
게임과금: 12,000원 (30%)
문구류: 8,000원 (20%)
옷: 5,000원 (12.5%)

균형이 좋습니다!
다양한 필요를 고르게 충족하고 있습니다."
```

### 2.5 반복 구매 비율 (Repeat Purchase Rate)

**정의**: 같은 상품/서비스를 반복적으로 구매하는 비율
**측정**: 월별 같은 항목 구매 횟수

**발달단계별 기준값**:

| 발달단계 | 반복 구매 상위 항목 | 빈도 | 해석 |
|---------|-----------------|------|------|
| **초등 저학년** | 간식 | 주 2-3회 | 충동적, 필요성 판단 낮음 |
| **초등 고학년** | 간식 | 주 1-2회 | 습관적이지만 조절 시작 |
| **중학생** | 간식, 문구류 | 주 1회 | 필요 기반, 계획적 |
| **고등학생** | 특정 브랜드, 서비스 | 월 2-4회 | 선호도 기반, 신중한 선택 |

**아이부자 활용**:
```
"이번 달 반복 구매 분석:
- 편의점 간식: 8회 (과다)
- 카페 음료: 4회 (정상)
- 게임 과금: 3회 (정상)

간식 구매가 너무 자주입니다.
대신 다양한 음식을 시도해보세요!"
```

## 3. 시간 기반 지표

### 3.1 저축 지속 기간 (Savings Duration)

**정의**: 특정 목표를 위한 저축이 얼마나 오래 유지되는가
**측정**: 목표 설정부터 달성 또는 포기할 때까지의 기간

**발달단계별 기준값**:

| 발달단계 | 달성 가능 기간 | 평균 달성률 | 해석 |
|---------|--------------|----------|------|
| **초등 저학년** | ~1개월 | 30% | 단기 목표만 가능, 자주 포기 |
| **초등 고학년** | 1-3개월 | 60% | 3개월 목표는 달성 가능 |
| **중학생** | 3-6개월 | 70% | 중기 목표 달성 능력 |
| **고등학생** | 6-12개월 | 75% | 장기 목표 유지 능력 |

**근거**: Mischel과 Ayduk (2011)의 연구에서 지연만족 능력이 청소년기에 급격히 발달하며, 이것이 저축 지속 능력으로 직결된다고 보였습니다.

**아이부자 추적 방식**:

```
목표 추적 타임라인:

목표: "여름 휴가 용돈 100,000원"
설정: 2025-01-15
진행:
├─ 2025-01-15: 5,000원 저축 시작
├─ 2025-02-15: 10,000원 저축 (진행률 10%)
├─ 2025-03-15: 20,000원 저축 (진행률 20%)
├─ 2025-04-15: 30,000원 저축 (진행률 30%) ← 지금
└─ 예상 달성: 2025-08-15 (8개월)

지속 기간: 3개월 유지 ✓
위험도: 낮음
피드백: "3개월을 꾸준히 진행했습니다!
        목표를 잘 유지하고 있습니다."
```

### 3.2 거래 빈도 (Transaction Frequency)

**정의**: 월간 얼마나 자주 거래(구매 또는 저축)를 하는가
**측정**: 월별 거래 건수

**발달단계별 기준값**:

| 발달단계 | 월간 거래 빈도 | 일일 평균 | 해석 |
|---------|-----------------|----------|------|
| **초등 저학년** | 20-30회 | 1-1.5회/일 | 충동적, 자주 구매 |
| **초등 고학년** | 15-25회 | 0.5-0.8회/일 | 계획적 사용으로 감소 |
| **중학생** | 12-18회 | 0.4-0.6회/일 | 신중한 선택 |
| **고등학생** | 8-12회 | 0.3-0.4회/일 | 목적 지향적 구매 |

**중요 인사이트**:
- **감소 추이**: 거래 빈도가 감소 = 충동 조절 능력 향상
- **증가 추이**: 거래 빈도 증가 = 또래 영향이나 스트레스 받음
- **급격한 변화**: 보상이나 스트레스 상황 발생 가능성

**아이부자 활용**:
```
"월간 거래 분석:
- 1월: 28회 (높음)
- 2월: 22회 ▼
- 3월: 18회 ▼
- 4월: 15회 ▼

충동 조절이 점점 좋아지고 있습니다!"
```

### 3.3 구매 전 대기 기간 (Pre-Purchase Waiting Period)

**정의**: 물건을 "사고 싶다"고 생각한 후 실제 구매까지의 기간
**측정**: 온라인 장바구니 보관 시간 또는 앱 내 "보류" 기능 사용

**발달단계별 기준값**:

| 발달단계 | 평균 대기 기간 | 최대 대기 기간 | 해석 |
|---------|---------------|---------------|------|
| **초등 저학년** | 즉시 (0일) | 1-2일 | 거의 충동적 구매 |
| **초등 고학년** | 1-2일 | 5-7일 | 간단한 검토 과정 |
| **중학생** | 3-5일 | 10-14일 | 신중한 재고 |
| **고등학생** | 1주일 이상 | 3주 이상 | 깊이 있는 검토 |

**아이부자 활용**:

아이부자 내 "관심 목록" 또는 "장바구니" 기능:
```
관심 목록: 게임 기기 (50,000원)
├─ 추가한 날: 2025-03-15
├─ 대기 기간: 23일
├─ 상태: 아직 구매 안 함
└─ 피드백: "3주를 검토했군요.
           정말 필요한지 다시 한 번
           생각해보세요."

결과:
- 구매한 경우 → "충동적 구매를 계획적으로 변경했어요!"
- 포기한 경우 → "신중한 판단 능력이 좋네요!"
```

## 4. 패턴 기반 지표

### 4.1 부모 상담 비율 (Parent Consultation Rate)

**정의**: 전체 거래 중에서 부모의 승인/상담을 거친 비율
**측정**: "부모에게 물어보기" 기능 사용 비율

**발달단계별 기준값**:

| 발달단계 | 부모 상담 비율 | 상담 대상 거래액 | 해석 |
|---------|-------------------|------------|------|
| **초등 저학년** | 80-100% | 전체 거래액 | 부모 의존적 |
| **초등 고학년** | 50-70% | 10,000원 이상 | 부분적 독립 |
| **중학생** | 20-40% | 20,000원 이상 | 대부분 자율 결정 |
| **고등학생** | <20% | 30,000원 이상 | 거의 완전 자율 |

**아이부자 추적**:

```
거래 상세 분석:
총 거래 수: 18회

1. 간식 (3,000원): 부모 상담 안 함 ●
2. 게임과금 (5,000원): 부모 상담 함 ▲
3. 책 (12,000원): 부모 상담 함 ▲
4. 신발 (25,000원): 부모 상담 함 ▲
...

부모 상담 비율: 33% (6/18회)
```

### 4.2 거래 실패/반품 비율 (Transaction Regret Rate)

**정의**: 구매 후 반품, 환불, 또는 "이걸 사지 말았으면" 평가한 비율
**측정**: 월간 반품 건수 / 전체 구매 건수

**발달단계별 기준값**:

| 발달단계 | 반품률 | 해석 |
|---------|--------|------|
| **초등 저학년** | 20-40% | 충동 구매가 많음 |
| **초등 고학년** | 10-20% | 계획성 향상 |
| **중학생** | 5-10% | 신중한 결정 |
| **고등학생** | <5% | 높은 의사결정 능력 |

**근거**: 이 지표는 계획적 구매 능력과 충동조절 능력을 함께 반영합니다.

**아이부자 활용**:
```
"이번 달 구매 평가:
총 구매: 15회
만족: 12회 (80%)
후회: 3회 (20%)

'이걸 사지 말았으면' 했던 항목:
- 게임 과금 (충동적)
- 유행 악세서리 (친구 따라함)
- 다이어트 식품 (실제로 안 먹음)

팁: 게임 과금과 유행 상품은
     48시간 대기 규칙을 적용해보세요!"
```

## 7. 서비스 기획 implications

### 7.1 상호작용 패러다임 자동 전환 트리거

아이부자는 다음의 조건을 만족할 때 자동으로 부모에게 "상호작용 방식 전환 권고"를 제시할 수 있습니다.

**초등 고학년 → 중학생 전환 신호**:
```
다음 조건 중 3개 이상 충족:
□ 저축률이 20% 이상 유지 (2개월)
□ 용돈 소진 기간이 10일 이상 (계속)
□ 부모 상담 비율이 60% 이하
□ 저축 목표 3개월 이상 지속
□ 반품률이 20% 미만

→ "상호작용 패러다임 전환"을 권고하는
   알림 발송
```

### 7.2 커스터마이즈된 코칭 메시지

각 아동의 거래 데이터를 기반으로 AI가 자동으로 생성하는 개인화된 피드백:

```
예 1) 저축률은 높지만 거래 빈도가 많은 경우:
"저축은 잘 하고 있지만, 거래가 자주 일어나고 있어요.
 각 구매를 더 신중하게 생각해보면 어떨까요?
 '48시간 대기 규칙'을 시도해보세요!"

예 2) 반품률이 높은 경우:
"반품한 상품의 대부분이 '친구가 하니까' 구매한 것들이군요.
 친구의 영향은 중요하지만, 너의 필요도 중요해요.
 다음에는 정말 필요한지 부모와 한 번 상의해보세요!"

예 3) 저축률이 정체된 경우:
"지난 2개월 저축률이 20% 정도로 정체되었어요.
 다음 단계(중학생 수준 35%)에 도달하려면,
 목표를 더 크게 설정해보는 건 어떨까요?"
```
