## 1단계. 7~9세 (관찰자)

> 부모가 모든 금융 행위를 대리. 자녀는 결과를 보고 관찰하는 역할.  
> 이 시기의 핵심은 "돈이 오고 가는 것을 눈으로 보는 경험"

### ⚖️ 법적 가능 범위
- 예·적금 **입금**: 자녀 단독 가능 (권리만 얻는 행위, 민법 제5조 단서)
- 예·적금 **해지·인출**: 친권자 동의 필요
- 증권계좌 개설: 대면 개설만 가능 (비대면 불가)
- 매매 실행: 부모 전부 대리
- 증여 수령: 자녀 단독 가능

### 📱 자녀 앱

**As-is 기능**
- 지갑 > 내 지갑 (잔액 확인)
- 용돈관리 > 용돈계획 (용돈 현황 보기)
- 용돈관리 > 거래내역
- 용돈 활동 > 저금 (저금통 보기)
- 용돈 활동 > 미션 (오늘의 할 일 확인)
- 오늘의 활동 > 오늘의 기분, 걷기 챌린지, 퀴즈 풀기, 습관 만들기

**To-be 기능**
- 지갑 / 계좌 / 저금통 역할을 명확히 구분 (저금통 UI 중심) `#서브`
- 연령별 3단계 UI 분기 — **저금통 개념** 화면 `#서브`
- 소비 후 칭찬 or 넛지 메시지 (간단한 수준) `#서브`
- 오늘의 기분 등 활동을 통한 포인트 적립 경험

### 👨‍👩‍👧 부모 앱

**As-is 기능**
- 지갑 > 내 지갑 (자녀 잔액 확인)
- 용돈관리 > 용돈 지급 (용돈 주기)
- 용돈관리 > 정기용돈 (정기 용돈 설정)
- 용돈관리 > 활동 보고서
- 용돈 활동 > 미션 (미션 설정)
- 용돈 활동 > 저금 (저금 현황 확인)

**To-be 기능**
- 입학·생일 등 생애 이벤트 기반 가입 알림 `#서브`
- 부모 대상 "아이 금융 첫걸음" 선택 콘텐츠 `#서브`
- 부모 대상 "자녀 연령별 적정 용돈 금액" 데이터 기반 가이드 `#서브`
- 매칭저축 & 미션저축 구조 — 부모가 조건 설계 `#메인`
- 매칭저축 계좌 (자녀 저축 → 부모 매칭 자동 입금 구조) `#메인`

### 🆕 이 단계에서 새로 열리는 것
- 아이부자 서비스 최초 가입 (부모 주도)
- 저금통 개념의 첫 저축 경험
- 미션 수행 → 용돈 수령의 인과관계 경험
- 출생 기념 씨앗통장 개설 가능 (하나은행 통장) `#서브`

## 2단계. 10~13세 (선택자)

> 부모가 허락한 범위(민법 제6조) 안에서 자녀가 선택하기 시작.  
> 카드 사용, 송금 등 실제 소비 행위가 시작되는 시기.

### ⚖️ 법적 가능 범위
- 부모가 범위를 정해 허락한 재산: 자녀 임의 처분 가능 (민법 제6조)
- 예·적금 입금: 단독 가능
- 카드 결제·송금: 부모 사전 허락 범위 내 가능
- 증권계좌 개설: 대면만 가능 (비대면 불가, 14세 미만)
- 주식 매매: 부모 대리 실행 (자녀는 선택지 제안)
- 증여 수령: 단독 가능

### 📱 자녀 앱

**As-is 기능**
- 지갑 > 내 지갑, 카드관리, 충전계좌관리
- 용돈관리 > 용돈계획, 용돈요청, 거래내역, 활동 보고서
- 용돈 사용 > 카드결제, 제로페이, 송금, 송금내역, 더치페이
- 용돈 활동 > 미션, 저금, 기부
- 오늘의 활동 > 오늘의 기분, 걷기 챌린지, 퀴즈 풀기, 습관 만들기, 자원봉사, 드림어필

**To-be 기능**
- 연령별 3단계 UI 분기 — **계좌 개념** 화면 전환 `#서브`
- 소비 성공/실패 경험 가이드 `#메인`
- 소비 예측 알림 (e.g. '이렇게 쓰면 에어팟까지 6개월 걸려요') `#서브`
- 구매 목표 기반 미션 저축 연계 `#서브`
- 소비 행동을 칭찬하고, 남은 용돈을 적절히 활용할 수 있도록 가이드 제공 `#서브`
- 실패 경험 → "다음엔 이렇게" 복기 가이드 `#서브`
- 과소비 감지 시 절약 챌린지 제안 `#서브`
- 매칭저축 장기 유지를 위한 중간 보상 (구간별 달성 뱃지 등) `#서브`
- 연령별 수령 개념 설명 콘텐츠 — 계좌 개념 `#서브`
- 미션 달성으로 인한 항돈 수령을 소수 ETF 주식으로 변환 (시드 없이 투자 경험) `#서브`

### 👨‍👩‍👧 부모 앱

**As-is 기능**
- 지갑 > 내 지갑, 충전계좌관리
- 용돈관리 > 용돈 지급, 정기용돈, 거래내역, 활동 보고서
- 편리사용 > 송금, 더치페이
- 용돈 활동 > 미션, 저금

**To-be 기능**
- 용돈 인상 요청 승인/거절 기능 `#메인`
- 소비 예측 알림 (부모에게도 알림) `#서브`
- 유사 연령대·거주지 기반 용돈 통계 넛지 `#서브`
- 금융 신용점수 기반 용돈 조정 제안 `#서브`
- 저축 / 소비 / 투자 배분 교육 및 가이드 `#서브`
- 금융 교육 콘텐츠 — 금융 용어/금융 상식 맥락형 교육 (부모 대상) `#메인`

### 🆕 이 단계에서 새로 열리는 것
- **카드 결제 / 송금** 기능 오픈 (부모 허락 범위 내)
- **용돈 요청** 기능 (자녀가 직접 요청)
- 소비 경험 시작 → 소비 피드백 기능 활성화
- 청소년 전용 체크카드 발급 가능 `#서브`
- 입학 기념 계좌 개설 패키지 (우대금리) `#서브`

## 3단계. 14~15세 (제안자)

> 비대면 계좌 개설이 가능해지는 법적 전환점.  
> 자녀가 투자 제안을 직접 작성하고, 부모가 개별 동의하는 구조.

### ⚖️ 법적 가능 범위
- **비대면 계좌 개설 가능** (일부 은행·증권사, 친권자 전자 동의)
- 증권계좌 개설: 비대면 가능 (친권자 전자 동의)
- 주식·ETF 매수: 부모 개별 동의 후 실행
- 자녀가 투자 제안 작성 → 부모 개별 승인 → 부모 명의로 실행
- 포괄 허락 범위 안에서 자율 소비 확대 (민법 제6조)

- 용돈 인상 요청 (기존 아이부자 '계획하기' 기능 활용) `#메인`
- 연령별 3단계 UI 분기 — **자산 개념** 화면 전환 `#서브`
- 시장 이벤트 발생 시 즉시 교육 연결 `#서브`
- 금융 교육 콘텐츠 — ETF, ISA 등 맥락형 교육 (자녀 대상 시작) `#메인`
- AI금융 코치 캐릭터 도입 (패턴 분석 → 피드백) `#서브`

## 4단계. 16~18세 (운용자)

- 레버리지·파생상품: 여전히 제한 (고위험)
- 성인 금융 상품 예고 시작 가능

## 5단계. 19세+ (독립 견습 → 성인)

> 민법 제4조 성년 도달. 법적 대리 관계 완전 해소.  
> 하나원큐로 자산·이력 이전. 부모는 조언 관계로 전환.

### ⚖️ 법적 가능 범위
- **모든 금융 행위 단독 가능** (민법 제4조)
- 법정대리인 동의 불필요
- 전문투자자 요건 충족 시 고위험 상품 접근 가능
- 성인 증여 비과세 한도 적용 (직계존속 10년간 5천만원)

### 📱 자녀 앱 → 하나원큐 전환

**As-is 기능**
- (아이부자 서비스 종료 → 하나원큐 이관)

**To-be 기능**
- 하나원큐 연동 — 금융 이력 이전 `#서브`
- 성인 금융 프리뷰 교육 (청년도약계좌, 신용카드 등) `#서브`
- 아이부자에서 쌓인 미성년자 신용점수 → 성인 신용점수 연계 `#메인`
- 데이터 기반 금융 히스토리 — 하나원큐로 완전 이전 `#메인`

### 👨‍👩‍👧 부모 앱

**To-be 기능**
- 부모 대상 자녀 성인 이후 절세 정보 콘텐츠 `#서브`
- 절세/증여 관련 상품 (성년 자녀 증여 한도 5천만원으로 확대) `#메인`
- 은행 상품 연계 (주택청약·ISA 전환 패키지 등) `#메인`

### 🆕 이 단계에서 새로 열리는 것
- **하나원큐 자동 이관** (금융 이력 + 자산 전체)
- **법적 대리 해소** — 단독 금융 행위 전면 가능
- 성인 증여 비과세 한도 **5천만원**으로 확대
- **청년도약계좌** 등 성인 전용 상품 접근
- 장학금 연계 적금 만기 수령
