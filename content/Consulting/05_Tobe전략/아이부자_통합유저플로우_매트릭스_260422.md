**작성일**: 2026-04-22
**버전**: v1.0
**작성**: VINYLC / Claude (Cowork)
**목적**: 6개 스테이지 × 8개 레이어로 구성된 통합 사용자 여정의 전략적 내용 구조화
**참조 소스**: 전략장표 내러티브(260420), 진입이용유도설계(260421), 아이디어 통합정리(260422), 통합플로우 drawio v2.0

## Part 2. 스테이지별 상세

---

### S0. 부모 준비기 (출생~6세)

> **부모 감정**: "아이에게 금융 교육을 시켜야 한다는데, 어디서 어떻게 시작하지?"
> **서비스 가치**: 부모의 막연한 의무감을 구체적인 첫 행동으로 전환

#### ① 법적 프레임 & 부모 역할

| 항목 | 내용 |
|------|------|
| 적용 법조항 | 민법 제920조 (친권자 완전 대리) |
| 부모 역할 | 🛡 **설계자** — 계좌 개설, 투자 설정, 시드 구조를 전부 직접 관리 |
| 자녀 법적 가능 범위 | 없음 (의사무능력자, 7세 미만) |
| 자녀 서비스 인지 | 서비스를 직접 경험하지 않음 |

#### ② 자녀 핵심 경험 목표

이 단계에서 자녀는 앱을 사용하지 않습니다. 대신 **일상 속에서 돈에 대한 원초적 감각**을 형성합니다.

| 연령 구간 | 인지발달 특성 | 경험 목표 |
|-----------|-------------|-----------|
| 0~3세 | 전조작기 (피아제) — 수 개념 미형성 | 부모의 돈 이야기를 자연스럽게 듣는 환경 |
| 4~6세 | 전조작기 후반 — '더 많은 것' 인지 시작 | "이것 사면 저것 못 사" 기회비용 첫 체감, 동전 모으기 습관 |

#### ③ 자녀 앱 내 액션 시퀀스

**해당 없음** — 자녀는 이 단계에서 앱을 사용하지 않습니다.

자녀의 실생활 경험만 존재합니다:
- 심부름 후 용돈 수령 → 다 써봄 (소비의 원체험)
- 우연히 남김 → 부모의 칭찬 → "아끼면 좋은 일이 생기는구나" 첫 인식
- 마트/편의점에서 금액 비교 체험 ("이건 3천원, 이건 5천원")

#### ④ 부모 앱 내 액션 시퀀스

| 순서 | 행동 | 화면/기능 | 비고 |
|------|------|-----------|------|
| 1 | 아이부자 인지 | PB 권유, 세뱃돈 계기, 주변 추천 | 진입 채널 다양화 |
| 2 | 앱 설치 & 가입 | [화면] 가입·계좌 개설 마법사 | 자녀 나이 입력 → 단계 자동 매핑 |
| 3 | 투자계좌 A 개설 | [화면] ETF 자동매수 설정 | 자녀 나이 기반 ETF 추천, 자동매수 월 1회 기본 |
| 4 | 선불충전 지갑 개설 | [화면] 용돈 전달용 지갑 | 향후 용돈 지급 준비 |
| 5 | 세뱃돈·목돈 입금 | 이벤트 시드 계좌 입금 | 설날/추석/생일 시 자동 안내 |
| 6 | 성장 공유 확인 | 월간 투자 수익률 확인 | "이번 달 +2.3%, 아이에게 보여줄까요?" |

#### ⑤ 시드 구조 & 계좌 운용

| 항목 | 상태 |
|------|------|
| 활성 계좌 | 투자계좌 A (부모 전적 관리) |
| 시드 채널 | 이벤트 시드만 활성 (세뱃돈·목돈·조부모 선물금) |
| 매칭 | 미활성 (자녀 저축 행동이 없으므로) |
| 용돈 주기 | 미시작 (간헐적 현금 용돈만) |
| 출생 증여 유도 [A-4] | 10년 주기 2천만원 비과세 증여 제도 안내 → 하나은행에서 운용 유도 |

#### ⑥ 앱 외 상호작용

| 접점 | 내용 |
|------|------|
| **부모-자녀 대화** | 자녀 앞 돈 이야기 자연화, 저금통 놀이 시작 |
| **조부모·친척** | 세뱃돈/생일 선물금 수령 → 부모가 시드계좌 입금 |
| **PB·지점** | PB 미팅: "자녀 계좌도 함께 만드실래요?", 부모 금융상품 가입 시 자녀 계좌 번들 |
| **학교·또래** | 해당 없음 |

#### ⑦ 프론트 시스템 (채널×메시지)

**하나원큐 채널**

| 트리거 | 메시지 |
|--------|--------|
| 설날 D-7 AND 미가입 AND 하나원큐 고객 | "설날 세뱃돈, 주변 부모들은 이미 아이 계좌에 넣고 있어요" |
| PB 미팅 예약 완료 | PB 대면: "자녀 계좌도 함께 만드실래요?" |
| 조부모 선물 시즌 | "아이 이름 계좌에 1만원이 10년 후 얼마가 될까요?" |

**아이부자 채널**

| 트리거 | 메시지 |
|--------|--------|
| 가입 완료 | 초기 설정 마법사: 자녀 나이 기반 ETF 추천 + 자동매수 월 1회 기본 |
| 세뱃돈 입금 감지 | "세뱃돈 50,000원이 투자계좌에 들어갔어요" |
| 월 1회 수익률 계산 | "이번 달 계좌가 +2.3% 불어났어요. 아이에게 보여줄까요?" + 복리 성장 시각화 |

**기타 채널 (카카오/SMS)**

| 트리거 | 메시지 |
|--------|--------|
| 미가입 AND 하나원큐 고객 | "아이 용돈, 그냥 주면 그냥 써요. 교육적으로 줄 수 있는 방법 있을까요?" |

**부모 교육 콘텐츠**

| 콘텐츠 | 내용 |
|--------|------|
| '부모의 돈 대화법' 5분 영상 | 자녀 앞에서 돈 이야기를 자연스럽게 하는 방법 |
| 연령별 용돈 적정 금액 가이드 | 지역·연령별 데이터 기반 제안 |
| 자녀 금융교육 체크리스트 | 0~6세 필수 경험 항목 |
| 법적 안내 | 미성년자 계좌개설 요건, 연간 증여세 면제 2천만원, 법정대리인 동의 범위 |

#### ⑧ 성공 지표 & 전환 조건

**핵심 KPI**

| 지표 | 목표 |
|------|------|
| 계좌 개설 완료율 | 가입 시작 대비 A계좌 개설 완료 ≥ 80% |
| 이벤트 시드 1회 이상 입금 | 가입 후 6개월 내 ≥ 60% |
| 부모 월간 재접속률 | ≥ 40% |

**수집 데이터**: 가입 유입 채널, 가입 시 자녀 나이, 진입 계기 유형, 시드계좌 초기 금액, ETF 자동매수 주기, 설정 완료 여부, 공유 클릭 여부

**→ 게이트 0→1 전환 조건**: 자녀 나이 7세 도달 AND (부모가 자녀 앱 설치 완료 OR 자녀 첫 로그인)

### S1. 금융 감각 형성기 (7~9세)

> **부모 감정**: "용돈을 그냥 주면 다 써버리는데. 저축하라고 하면 잔소리가 되고."
> **서비스 가치**: 부모의 잔소리를 시스템으로 대체 — 자연스러운 저축 습관 형성

#### ① 법적 프레임 & 부모 역할

| 항목 | 내용 |
|------|------|
| 적용 법조항 | 민법 제920조 (친권자 완전 대리, 계속) |
| 부모 역할 | 👁 **코치** — 아이와 함께 앱을 보며 관심을 이끌고, 매칭·이벤트 시드 관리 |
| 자녀 법적 가능 범위 | 입금(권리만 얻는 행위) 단독 가능, 해지·인출 불가 |
| 부모 감정·동기 | 🔧 "좋은 습관의 구조를 만들어준다" |

#### ② 자녀 핵심 경험 목표

| 경험 시퀀스 단계 | 도달 수준 |
|-----------------|-----------|
| ① 시드 생성 | 세뱃돈·용돈으로 첫 시드 형성 |
| ② 투자 | 부모와 공동 첫 투자 결정 |
| ③ 돈이 굴러가는 체험 | ⭐ "어? 돈이 늘어나 있네?" 배당금/이자 발견 |
| ④ 학습 | 아직 체계적 학습 전, 감각적 인지 수준 |

**인지발달 특성 (6~8세)**:
- 구체적 조작기 초입 (피아제)
- '더 많은 돈 = 더 많은 것' 이해 시작
- 미래 개념 미약 → 시각적 피드백 필수
- 충동 소비 높음 — 숫자보다 시각 성장으로 접근

**이 단계 달성 기준**:
- 첫 저축 목표 달성 경험 (예: 8천원 장난감 8주)
- "내 돈이 자란다" 첫 인식
- 월 저축률 5~20% 도달

#### ③ 자녀 앱 내 액션 시퀀스

**투자 트랙**

| 순서 | 행동 | 화면/기능 | 감정 변화 |
|------|------|-----------|-----------|
| 1 | 부모 권유로 앱 설치·첫 로그인 | [화면] 캐릭터 선택 + 첫 미션 온보딩 | 호기심 |
| 2 | 세뱃돈/용돈 수령 확인 | [화면] 잔액 확인 화면 | "내 돈이 여기 있네" |
| 3 | 부모가 모아온 시드 공개 확인 | 세뱃돈·목돈 누적 시각화 | 놀라움 |
| 4 | ⭐ 패시브 인컴 알림 수신 | "네 돈이 340원 늘었어" [A-6] | "어? 돈이 늘어나 있네?" |
| 5 | 조부모·친척 선물금 유입 | "돈이 점점 모인다" 알림 | 기대감 형성 |
| 6 | 부모 투자 포트폴리오 일부 공개 | [화면] 부모 투자 내역 일부 열람 | 투자 흥미 유도 |
| 7 | ⭐ 부모-자녀 공동 첫 투자 결정 | [화면] "같이 골라보자" ETF 선택 | "내가 골랐다!" |

## Part 4. 횡단 시스템 (전 스테이지 공통)

### 4-1. 패시브 인컴 트래커 [A-6]

S1부터 S5까지 전 연령 작동. "자는 동안 네 돈이 일했어요 — +340원". 배당·이자 발생 시 자녀 앱에 누적 시각화. "내가 지금까지 받은 패시브 인컴 총액" 항상 표시.

### 4-2. 내 투자 이야기 연대기 [A-7]

계좌 생성일부터 현재까지 타임라인 시각화. "7살 때 첫 배당금 340원, 지금은 월 3,200원". S5 이관 시 Graduation 페이지의 핵심 콘텐츠.

### 4-3. 졸업 카운트다운 [A-10]

"지금 이 속도로 계속하면 18세에 ○○만 원 예상" 항상 표시. S2부터 활성화.

### 4-4. AI 금융 코치 캐릭터 [C-1]

자녀 소비 패턴 분석 → 매주 미션 자동 생성 + 대화형 피드백. AI 대화형 설정 UI [C-3] — "나 뭐 사고 싶은데 어떻게 해야 되지?" → 계획 자동 생성.

### 4-5. 부모 AI 어시스턴트 [C-2]

"이 나이 평균 저축률" 비교, 또래 대비 자녀 금융 성장도 시각화. 금액이 아닌 용돈 대비 저축률/소비 대비 저축률로 표시.

### 4-6. 자산 연동 앱 테마 (성장형 비주얼) [G-4]

총 자산 규모에 따라 앱 배경·아이콘 자동 변화: 10만원 미만=새싹 → 50만원=나무 → 100만원=숲 → 500만원=도시 → 1,000만원=금융 제국. 자산 감소 시 시각적 축소.

### 4-7. 프로필 커스터마이징 & 돈 인터랙션 [E-2, E-3]

카톡처럼 프로필 꾸미기 — 모은 자산이 없을 때 심플 → 부유해질수록 리치한 아이콘 진화. 송금 시 금액이 "팍삭삭 무너져 날아가는" 인터랙션, 들어올 때 반짝반짝.

### 4-8. 연령별 메인 화면 변화 (Complexity Growth) [D-4]

어릴 때 저금통+용돈만 → 나이 들수록 자산·주식·투자 방법 점차 노출 → 19세에는 완전한 자산 관리 화면. "이 화면 변화를 만들어내는 서비스."

### 4-9. 부모 반복 트리거 5가지 [F-1]

| # | 트리거 | 유형 |

## 부록: 용돈 주기 성장 로드맵

```
[S0] 미시작 → [S1] 간헐적 → 일별 → [S2] 일별 → 주별 → [S3] 주별 → 월별 → [S4] 월별 → 분기별 → 반기별 → 학제별 → [S5] 종료/이관
```

각 전환은 자녀의 연속 달성 실적 + 부모 신뢰 지표에 기반하며, 서비스가 전환 시점을 제안하고 부모가 최종 결정합니다.
