- **설계 원칙**: 투자 경험 → "나는 돈을 키우는 사람"이라는 정체성 형성

### G-3. 실시간 자산 성장 피드백
- **내용**: 저축/투자 직후 즉시 "네 총 자산이 방금 124,300원 → 127,300원이 됐어!" 피드백 + 자산 성장 그래프 실시간 반영. 매칭이 붙으면 "엄마가 3,000원 보태서 130,300원!" 추가 표시
- **출처**: [D]
- **설계 원칙**: 작은 행동의 결과가 즉시 눈에 보여야 반복 동기가 생김

### G-4. 자산 연동 앱 테마 (성장형 비주얼)
- **내용**: 총 자산 규모에 따라 앱 배경·아이콘·잠금화면 위젯이 자동 변화. 예: 10만원 미만 = 새싹 → 50만원 = 나무 → 100만원 = 숲 → 500만원 = 도시 → 1,000만원 = 금융 제국. 자산 감소 시 시각적으로도 축소 (돈 쓰면 나무가 줄어드는 효과)
- **출처**: [D][M]
- **설계 원칙**: 자산 누적이 시각적 정체성의 일부가 되면 이탈 비용이 높아짐
