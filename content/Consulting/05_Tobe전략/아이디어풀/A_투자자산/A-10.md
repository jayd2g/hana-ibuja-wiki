# A-10. 졸업 카운트다운

**카테고리**: A. 투자·자산 관련
**출처**: [W-시드머니리서치][W-진입이용유도]
**요약**: 지금 이 속도로 계속하면 18세에 얼마가 됩니까 — 항상 표시

---

## 개요

홈 화면에 "지금 속도로 가면 18세에 OO만원"을 항상 표시한다. 성인이 되는 날이 구체적인 목표가 되고, 하나은행 성인 상품으로의 자연스러운 연결이 완성된다.

## 표시 예시

```
🎓 성인까지 D-1,247일
지금 속도로 가면 → 18세에 약 847만원
조금 더 모으면   → 18세에 약 1,200만원
```

## 전략적 의미

- **리텐션**: 성인이 되는 날이 목표 → 앱을 떠날 이유가 없음
- **크로스셀**: 성년 도달 시 하나원큐 ISA, 청년도약계좌로 자연 연결
- **부모 설득 도구**: "이 속도면 18세에 800만원 시드머니" → 부모 지속 참여 동기
