# B-4. 위시리스트 기반 저축 시뮬레이터

**카테고리**: B. 소비·저축 행동 관련
**출처**: [M]
**요약**: 물건 선택 → 목표 시기 설정 → 부모 기여 설정 → 월 저축액 자동 계산

---

## 개요

"자동차 살 때 옵션 설정하는 느낌"으로 저축 계획을 세운다. 페이지 단위 프로세스가 아니라 AI 대화처럼 자연스럽게 진행된다.

## 흐름 (예시)

```
뭘 갖고 싶어? → "에어팟 맥스"
가격은? → 앱이 자동 검색 또는 직접 입력 (450,000원)
언제까지 갖고 싶어? → "여름방학 전까지"
부모님께 얼마나 도움받을까? → 슬라이더로 설정 (최대 50%)

→ "매달 37,500원씩 모으면 7월 1일에 살 수 있어!"
```

## 설계 포인트

- 단계별 페이지 이동 없이 한 화면에서 모든 설정
- 부모 선수금 최대 50% 한도 자동 적용
- [[C-3]] AI 대화형 UI와 자연스럽게 결합 가능
