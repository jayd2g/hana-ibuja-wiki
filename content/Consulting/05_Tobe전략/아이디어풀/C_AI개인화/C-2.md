# C-2. AI 대화형 설정 UI
**카테고리**: C. AI·개인화 관련
**출처**: [M]
**요약**: 단계별 페이지 없이 AI와 대화하듯 계획 자동 생성
---
## 개요
"기능처럼 안 생겼으면 좋겠다"는 설계 원칙. 페이지를 넘기며 설정하는 방식이 아니라, AI와 대화하듯 자연스럽게 목표를 설정한다.
## 대화 예시
```
자녀: "나 에어팟 사고 싶은데"
AI: "얼마짜리야? 내가 찾아볼게"
자녀: "맥스 버전"
AI: "450,000원이네. 언제까지 사고 싶어?"
자녀: "여름방학 전에"
AI: "그럼 매달 56,000원씩 모으면 돼. 부모님한테 얼마 도움받을 거야?"
```
## 설계 원칙
- 서비스가 먼저 기본값 제안 → 자녀는 수정만 탭
- 한 화면에서 모든 설정 완료 (팝업으로 처리)
- "단계별 페이지는 답답하다" — 자연스러운 대화 흐름
