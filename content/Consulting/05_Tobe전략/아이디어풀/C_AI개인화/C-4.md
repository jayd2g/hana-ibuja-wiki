# C-4. 데이터 기반 용돈 제안

**카테고리**: C. AI·개인화 관련
**출처**: [D][M]
**요약**: 서비스가 먼저 나이·지역·패턴 기반 기본 용돈 세팅을 제안

---

## 개요

"송파구 사는 8살이면 용돈은 한 달에 한 번 이 정도씩 주시고..." — 서비스가 먼저 기본 세팅을 제안한다.

## 제안 예시

```
OO이 (만 8세, 서울 송파구) 맞춤 용돈 제안

💡 권장 용돈: 월 20,000원
💡 권장 주기: 주 1회 지급 (5,000원씩)
💡 저축 목표: 용돈의 20% (주 1,000원)

→ 이 설정을 기반으로 시작하고, 3개월 후 조정해보세요
[이대로 시작하기] [직접 설정하기]
```

## 데이터 기반

- 지역별 물가 수준
- 나이별 평균 용돈 데이터 (오픈서베이 등)
- 학제 단계별 권장 패턴
