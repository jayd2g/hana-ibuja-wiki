# D-4. 연령별 메인 화면 변화 (Complexity Growth)
**카테고리**: D. 권한 이양·성장 시스템
**출처**: [M]
**요약**: 나이 들수록 화면이 복잡해짐 — 19세에는 완전한 자산 관리 화면
---
## 개요
"이 화면 변화를 만들어내는 서비스가 돼야 한다" — 나이에 따라 앱 화면 자체가 달라진다.
## 단계별 화면 변화
| 나이 | 화면 |
|---|---|
| 7~9세 | 저금통 + 용돈 조르기 버튼만 |
| 10~12세 | 저금통 + 용돈 + 매칭 저축 목표 |
| 13~14세 | 저금통 + 용돈 + 매칭 + 투자 A/B 선택 |
| 15~16세 | 자산 현황 + 포트폴리오 + 제안 발송 |
| 17~18세 | 자산 관리 + 투자 자율 운용 + 세금 정보 |
| 19세+ | 완전한 자산 관리 화면 (하나원큐 연계) |
## 핵심 설계
- 화면이 "성장한다"는 느낌 → 더 복잡한 화면이 보상처럼 느껴짐
- 이탈 비용: "여기까지 왔는데 토스로 가면 처음부터 다시..."
