# 아이디어 의도별 묶음

**작성일**: 2026-04-23
**목적**: 각 아이디어가 어떤 행동/문제를 위한 것인지 의도 기준으로 재분류. 피그잼 재구조화의 기준으로 사용.
**제외**: C-1 AI 금융 코치 캐릭터, B-6 A/B 선택 카드, G-1 소비 전환 넛지, E-3 돈 나가는 인터랙션, A-5 물건→투자 단위 치환

---

## ① 저축/투자 행동을 했을 때 즉각 강화 — 반복하고 싶게 만들기

- **G-3** 실시간 자산 성장 피드백 → 저축 직후 숫자가 바뀌는 걸 즉시 보여줌. 행동 → 결과 연결
- **A-1** 저축 매칭 자동화 → 저축하면 부모 돈이 자동으로 붙어서 행동에 즉각 보상
- **B-8** 절약 감지 → 매칭 부스트 → 평소보다 덜 쓴 주를 자동 감지해 "이번 주 2.5배 찬스" 제공
- **E-5** 연속 달성 보상 → 꾸준히 하면 스트릭 보상. 끊기기 싫게 만들기
- **A-3** 포인트 → ETF 전환 → 미션/절약으로 쌓은 포인트를 실제 주식으로 변환. 행동의 결과가 실자산

---

## ② 원하는 것을 저축 목표로 바꾸기 — 소비 욕구를 동력으로

- **B-3** 위시리스트 → 매칭 목표 → 갖고 싶은 물건을 등록하면 저축 목표로 자동 전환
- **B-4** 저축 시뮬레이터 → "이렇게 모으면 몇 달 후 살 수 있어" 경로 시각화로 포기 방지
- **B-1** 소비 예측 알림 → "이렇게 쓰면 에어팟까지 6개월 더 걸려" — 현재 패턴의 결과를 미리 보여줌
- **B-5** 선수금 저축 게임화 → 부모가 먼저 돈을 걸고, 자녀가 나머지를 모으는 공동 미션 구조

---

## ③ 자산 성장의 장기 흐름을 눈에 보이게 — 오래 해야 할 이유 만들기

- **A-6** 패시브 인컴 트래커 → 이자·배당이 쌓이는 걸 항상 표시. "내가 돈을 버는 중"이라는 감각
- **A-7** 연대기 → 7살 첫 배당부터 지금까지 성장 히스토리 타임라인. 기록이 쌓일수록 이탈 비용 증가
- **A-10** 졸업 카운트다운 → "18세에 얼마가 됩니까"를 항상 표시. 성인 전환을 목표로 만들기
- **E-4** 히스토리 카드 → "첫 주식 수익", "100만 원 돌파" 등 마일스톤을 카드로 수집. 그만두면 이 컬렉션이 멈춤

---

## ④ "나는 투자자"라는 정체성 형성 — 앱과 자기 자신을 동일시하게

- **G-2** 투자자 명함 → 내 투자 프로필을 카드로 만들어 공유 가능. "나 이런 앱 써" 낙인 대신 자랑 포인트
- **E-2** 프로필 커스터마이징 → 자산 규모에 따라 캐릭터가 진화. 돈 쓰면 시각적으로 줄어듦
- **E-1** 배지/칭호 → "16살 기준 일론 머스크보다 부자" 같은 칭호. 금융 행동이 곧 정체성
- **G-4** 자산 연동 앱 테마 → 자산 규모가 앱 배경/아이콘으로 표현. 앱 자체가 내 자산의 거울

---

## ⑤ 이벤트성 돈을 자산 형성 기회로 연결 — 목돈이 생기는 순간을 포착

- **B-7** 세뱃돈 → 투자 전환 → 세뱃돈 받는 순간에 "절반만 투자하면 여름에 얼마" 시뮬레이션
- **A-4** 출생 증여 유도 → 태어나자마자 2천만 원 비과세 증여를 하나은행에서 운용하도록 유도
- **F-6** 증여 시뮬레이터 → 10년 단위 증여 한도 관리 + 세무사 연결. 부모가 증여 결정할 때 지원

---

## ⑥ 돈의 목적별 분리 구조 설계 — 소비/저축/투자가 섞이지 않게

- **A-2** 두 계좌 구조 → 투자계좌(장기 인출 제한) + 목표계좌(단기 물건 목표). UI에서 A가 더 크게 보임
- **A-1-1** 매칭 비중 연동 인출 제한 → 부모가 많이 매칭할수록 자금이 더 오래 묶임. 개입 수준 = 자금 성격
- **B-2** 용돈 분배기 → 용돈 수령 직후 소비/저축/투자 비율 자동 추천

---

## ⑦ 자녀 자율성 단계적 확대 — 신뢰가 쌓이면 권한이 늘어나는 구조

- **D-6** 5단계 권한이양 프레임 → 전체 서비스의 구조 뼈대. 단계가 올라갈수록 자녀 권한 확대
- **D-7** 단계 전환 위저드 + 레벨업 → 진학 등 전환 시 "새 권한 열렸어" 레벨업 경험
- **D-4** 연령별 화면 변화 → 어릴 때는 저금통, 클수록 투자/자산 화면으로 자연스럽게 복잡도 증가
- **D-1** 금융 신용 점수 → 저축률·목표 달성률 기반 점수. 점수가 오르면 자율 한도 자동 확대
- **D-2** 금융 계약서 → "몇 살까지 이만큼 모으면 용돈 목돈으로" — 공식 약속으로 신뢰 구조화
- **D-3** 연간 용돈 일괄 지급 → 계약 이행 시 용돈을 연 단위 목돈으로 전환. 자율성의 보상
- **D-5** 제안 투표 → 매칭금 일부를 자녀가 직접 배분 제안. 실제 의사결정 참여 경험
- **F-4** 사전 허락 리스트 → 부모가 카테고리별 한도 사전 설정 → 그 안에서 자녀가 자유롭게 소비

---

## ⑧ 시장·손실 경험을 금융 교육으로 전환

- **A-8** 시장 이벤트 연결 → 보유 종목 뉴스가 생기면 "오늘 네 주식이 +340원" 알림. 경제를 내 것으로 체감
- **A-9** 첫 손실 가이드 → 투자가 처음 마이너스 될 때 자동 교육 푸시. 이탈 대신 학습으로 전환
- **F-5** 가족 투자 회의 → 월 1회 가족이 함께 투자 현황 리뷰. 부모 포트폴리오 일부 공개(교육 소재)

---

## ⑨ 부모가 앱을 반복 방문할 이유 만들기

- **F-1** 부모 반복 트리거 5가지 → 자녀 인사이트 피드, 가족 자산 통합 뷰, 분기 리포트, 증여 도구, 자녀 성과 알림
- **F-2** 7가지 이벤트 트리거 → 세뱃돈 시즌, 배당 발생, 학제 전환 등 외부 이벤트를 앱 접점으로
- **C-2** 부모 AI 어시스턴트 → 또래 대비 자녀 성장도 시각화. 부모가 비교·확인하고 싶어서 다시 열게

---

## ⑩ 이탈 방지 — 그만두지 않게 막기

- **F-3** 중학교 이탈 방어 → 13세 이후 "감시앱" 낙인 방지, 소비 비공개 전환, 또래 아닌 가족 비교로 전환
- **F-7** 자동 이관 + 졸업 페이지 → 성년 되면 하나원큐로 자동 이관 + "18년 습관이 네 시스템" 졸업 경험

---

## ⑪ 초기 진입 마찰 최소화 — 시작하기 쉽게

- **C-3** AI 대화형 설정 UI → 페이지 단위 설정 대신 대화하듯이. 기능처럼 안 생기게
- **C-4** 데이터 기반 용돈 제안 → "송파구 8살이면 이 정도" 기본값 먼저 제안. 부모가 결정 부담 없이 시작
