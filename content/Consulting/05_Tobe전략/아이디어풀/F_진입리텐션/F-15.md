# F-15. 자동 이관 + Graduation 페이지

**카테고리**: F. 진입·리텐션 관련
**출처**: [W-진입이용유도]
**요약**: 성년 도달 시 하나원큐로 자동 이관 + 18년의 여정을 담은 졸업 페이지

---

## 개요

성년(18세)이 되는 날, 아이부자에서 하나원큐로 자동 이관이 이루어지며, 지금까지의 금융 여정을 담은 졸업 페이지가 제공된다.

## Graduation 페이지 구성

```
🎓 "18년의 습관이 이제 너의 시스템이 됩니다"

📅 첫 계좌 개설: 2024.03.01
💰 누적 매칭 총액: 3,240,000원
📈 총 투자 수익: +1,120,000원
🏆 달성 배지: 24개

첫 배당: 2025.03.15 (12원)
→ 오늘 월 배당: 8,400원

[하나원큐에서 계속하기]
```

## 이관 내용

- 기존 자산 데이터 완전 이관
- 투자 연대기 [[A-7]] 그대로 이관
- 하나원큐 ISA, 청년도약계좌 바로 연결
