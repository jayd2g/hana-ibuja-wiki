# F-10. 현실 효용

**카테고리**: F. 진입·리텐션 관련
**출처**: [W-진입이용유도]
**요약**: 자녀의 자산이 실제 현실에 어떤 효용을 가져오는지 매칭 표시

---

## 개요

자녀의 자산이 실제로 무엇을 살 수 있는지, 어떤 경험을 가능하게 하는지를 구체적인 사물·경험과 매칭해 표시한다. \"네 자산으로 닌텐도 스위치 OO개\", \"네 자산으로 일본 여행 항공권 OO장\" 식으로, 자산이 추상적 숫자가 아닌 \"가능한 경험\"으로 인식되도록 한다.

## 기능

1. 자녀 총 자산 변동 시 자동으로 현실 매칭 카드 생성
2. 자녀 위시리스트·관심 항목 데이터 활용해 매칭 대상 개인화
3. 매칭 단위 — 사물(닌텐도, 자전거 등), 경험(여행, 콘서트), 미래 자산(대학 등록금 OO%)
4. 자녀가 직접 \"이 항목으로 환산해보기\" 선택 가능

## 왜 효과적인가

청소년기 자녀에게 \"OO만원\"은 추상적이지만 \"닌텐도 스위치 2개\"는 구체적이다. 자산을 현실 효용으로 환산해 보여주면 저축·투자 행동의 의미가 직관적으로 이해되고, 사용 동기가 유지된다.
