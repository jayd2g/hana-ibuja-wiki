# F-8. 절세 게이지 시각화

**카테고리**: F. 진입·리텐션 관련
**출처**: [근거] · I-1
**요약**: 10년 단위 비과세 한도 사용률을 부모 메인 화면에 상시 노출하는 게이지

---

H년 단위 비과세 증여 한도(미성년 2,000만원·성년 5,000만원)의 사용률과 만료까지의 시간이 부모 메인 화면 상단에 게이지 바로 상시 노출된다. 한도를 놓치면 사이클 단위로 소멸되는 절세 기회를 부모가 잊지 않도록, 시뮬레이터에 진입하지 않아도 즉시 인지할 수 있는 형태로 정보가 제공된다.

## 기능

1. **부모 메인 화면 상단 게이지 바**
   ```
   [████████░░░░] 비과세 한도 1,400만원 / 2,000만원 (잔여 600만원)
   다음 리셋 D-365 (2027.04.27)
   ```
2. 한도 만료 90일 전 알림: "12월 31일 만료, 지금 입금 시 N% 절세"
3. 10년 누계 그래프: 두 사이클을 하나의 타임라인에
4. 분할 증여 제안: "잔여 600만원, 10·11·12월 분할 시 평가 변동 위험 분산"
5. [[A-13]] 주식 직접 증여 트리거: 잔여 한도 클릭 → 증여 흐름 진입

## 왜 효과적인가

부자 79%의 1순위 우려를 정면 해소. 한도 미사용 = 손실이라는 감각을 부모 화면 상시 노출로 만든다. 매번 시뮬레이터에 들어가지 않아도 실시간 인지 가능.

## 차별점

기존 [[F-6]] 증여 시뮬레이터는 사용자가 진입해야 하는 도구. 이 아이디어는 **상시 노출 게이지**로, 진입 없이도 자연스럽게 인식된다.
