# F-8. 또래→가족 비교 전환

**카테고리**: F. 진입·리텐션 관련
**출처**: [W-진입이용유도]
**요약**: 동년배 비교 지표를 \"또래\"에서 \"가족 단위\"로 전환

---

## 개요

사춘기 자녀에게 \"또래 평균 대비 너의 위치\" 같은 비교는 부담스럽다. 비교 기준을 \"가족 단위 자산 형성에 너의 기여\"로 전환해 비교 압박을 줄이고, 동시에 가족 정체성을 강화한다.

## 기능

1. 자녀 중학교 진입(만 13세) 시점부터 비교 지표 UI 자동 전환
2. \"또래 대비 저축률 OO%\" → \"가족 자산 중 너의 기여 OO원\" 식 표현으로 변경
3. 또래 데이터는 보조 지표로만 노출, 가족 데이터를 메인으로
4. 가족 목표(가족 여행, 자녀 대학자금 등) 대비 진척률 표시

## 왜 효과적인가

또래 비교는 \"내가 부족하다\"는 인식을 만들어 동기 저하로 이어질 수 있다. 가족 단위 비교로 전환하면 \"내가 가족에 도움이 된다\"는 긍정적 정체성이 형성되어 사용 동기가 유지된다.
