# F-13. 가족 투자 회의 템플릿

**카테고리**: F. 진입·리텐션 관련
**출처**: [W-진입이용유도]
**요약**: 월 1회 가족이 함께 투자 현황을 리뷰하는 템플릿 제공

---

## 개요

월 1회 가족이 함께 앉아 투자 현황을 리뷰하는 구조화된 템플릿을 제공한다. 부모 포트폴리오 일부를 자녀에게 공개해 교육 소재로 활용한다.

## 회의 템플릿 구성

```
📋 이번 달 가족 투자 회의

1. 이번 달 우리 집 자산 변화 (+/-%)
2. OO이 자산 변화 + 잘한 점
3. 부모님 포트폴리오 한 가지 공개
4. 다음 달 목표 설정

예상 소요시간: 20분
```

## 설계 원칙

- 부모가 먼저 자산을 공개 → 자녀의 개방 동기 강화
- 판단·평가 없이 현황 공유 → 심리적 안전감
- 월 1회의 리추얼 → 금융 대화를 가족 문화로
