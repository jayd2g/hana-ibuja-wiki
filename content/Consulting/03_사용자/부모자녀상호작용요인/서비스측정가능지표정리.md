# 아이부자 서비스 측정 가능 지표 통합 정리

## 개요

본 문서는 아이부자 플랫폼이 자동으로 추적할 수 있는 모든 행동 지표를 통합적으로 정리한 것입니다. 이 지표들은 자녀의 금융 발달 단계를 객관적으로 판단하고, 부모-자녀 상호작용 방식의 전환 시기를 결정하는 데 사용됩니다.

**핵심 원칙**:
- **객관성**: 거래 데이터에 기반한 수치만 사용
- **자동성**: 시스템이 자동으로 계산
- **실행성**: 부모 또는 자녀가 실제로 행동 변화를 이룰 수 있는 지표

---

## 목차

1. 지표 분류 체계
2. 거래 데이터 기반 지표
3. 시간 기반 지표
4. 패턴 기반 지표
5. 목표 기반 지표
6. 발달단계별 기준값 종합 표
7. 대시보드 설계 원칙
8. 실행 가이드

## 1. 지표 분류 체계

아이부자가 추적하는 지표는 4가지 카테고리로 분류됩니다:

```
┌─────────────────────────────────────────────┐
│    아이부자 시스템 측정 지표 (v2.0)         │
├─────────────────────────────────────────────┤
│                                              │
│ 1. 거래 데이터 기반 지표 (Transaction-based)│
│    - 저축률, 사용률, 카테고리 다양성       │
│                                              │
│ 2. 시간 기반 지표 (Time-based)              │
│    - 소진 기간, 저축 지속 기간             │
│    - 거래 빈도, 구매 대기 기간             │
│                                              │
│ 3. 패턴 기반 지표 (Pattern-based)          │
│    - 부모 상담 비율, 반품률                │
│    - 반복 구매 패턴                         │
│                                              │
│ 4. 목표 기반 지표 (Goal-based)             │
│    - 목표 설정 수, 달성률                  │
│    - 목표 지속 기간                         │
│                                              │
└─────────────────────────────────────────────┘
```

## 2. 거래 데이터 기반 지표

### 2.1 월간 저축률 (Monthly Savings Rate)

**정의**: 월간 용돈 중 저축으로 돌린 금액의 비율

**공식**: (월간 저축액 / 월간 용돈액) × 100%

**데이터 출처**: 거래 기록 > 저축 거래만 필터링

**아이부자 추적 방식**:
```
월별 자동 계산, 월별 변화 추이 그래프 표시
```

**발달단계별 기준값**:

| 발달단계 | 평균값 | 목표 범위 | 신호 |
|---------|--------|---------|------|
| **초등 저학년** | 5% | 0-15% | 저축 개념 형성 |
| **초등 고학년** | 20% | 15-35% | 저축 습관 형성 |
| **중학생** | 35% | 25-50% | 저축 의식 확립 |
| **고등학생** | 40% | 35-60% | 계획적 재무관리 |

**해석 가이드**:
- **증가 추세**: 정상적 발달 신호
- **정체**: 외부 개입 필요 신호
- **급락**: 스트레스 상황 또는 큰 구매 신호

---

### 2.2 월간 사용률 (Usage Rate)

**정의**: 월간 용돈 중 실제 사용액의 비율

**공식**: (월간 사용액 / 월간 용돈액) × 100%

**참고**: 저축률 + 사용률 = 100%

**발달단계별 기준값**:

| 발달단계 | 사용률 | 저축률 | 해석 |
|---------|--------|--------|------|
| **초등 저학년** | 95% | 5% | 거의 모두 사용 |
| **초등 고학년** | 80% | 20% | 일부 저축 |
| **중학생** | 65% | 35% | 저축 우선 의식 |
| **고등학생** | 60% | 40% | 계획적 관리 |

---

### 2.3 사용 금액의 카테고리 다양성 (Consumption Diversity)

**정의**: 월간 사용액이 얼마나 다양한 카테고리에 분산되어 있는가

**측정 방법**:
```
1. 모든 거래를 카테고리로 분류 (음식, 게임, 패션, 학용품 등)
2. 각 카테고리별 지출 비율 계산
3. 최상위 카테고리의 집중도 측정
```

**예시**:

```
낮은 다양성 (위험):
게임과금: 45%
간식: 35%
기타: 20%
→ 2개 카테고리에 80% 집중

중간 다양성 (정상):
간식: 25%
게임과금: 20%
학용품: 20%
옷: 15%
기타: 20%
→ 4개 이상 균형

높은 다양성 (건강):
다양한 5개 이상 카테고리에 균형 분산
```

**발달단계별 기준값**:

| 발달단계 | 카테고리 수 | 위험 신호 |
|---------|-----------|---------|
| **초등 저학년** | 1-2개 | 정상 (한 가지에 집중 가능) |
| **초등 고학년** | 2-3개 | 1개 >60% |
| **중학생** | 3-4개 | 1개 >50% |
| **고등학생** | 4개 이상 | 1개 >40% |

**아이부자 활용**:
```
"이번 달 카테고리 분석:

📊 간식: 18,000원 (45%)
🎮 게임: 14,000원 (35%)
📚 학용품: 8,000원 (20%)

⚠️ 간식 비중이 높습니다.
다양한 필요를 고르게 충족해보세요!"
```

---

### 2.4 반복 구매 비율 (Repeat Purchase Rate)

**정의**: 같은 상품/서비스를 반복적으로 구매하는 비율

**측정 방법**:
```
1. 같은 상품/가맹점에서의 중복 구매 추적
2. 월별 반복 구매 건수 계산
3. 상위 반복 항목 목록화
```

**예시**:

```
반복 구매 목록:
- 편의점 (OO과자): 8회/월
- 카페 (스타벅스): 4회/월
- 게임 과금: 3회/월

분석: 편의점 구매가 너무 자주 (평균 1.9회/주)
```

**발달단계별 기준값**:

| 발달단계 | 빈도 | 의미 |
|---------|------|------|
| **초등 저학년** | 주 2-3회 | 충동적, 조절 능력 낮음 |
| **초등 고학년** | 주 1-2회 | 습관적이지만 조절 시작 |
| **중학생** | 주 1회 | 필요 기반, 계획적 |
| **고등학생** | 월 2-4회 | 신중한 선택 |

## 3. 시간 기반 지표

### 3.1 용돈 소진 기간 (Days to Depletion)

**정의**: 용돈을 받은 후 완전히 소진될 때까지의 기간

**측정 방법**:
```
매월 1일: 용돈 수령
계산: 용돈 수령일 ~ 잔액 0이 될 때까지의 일수
```

**발달단계별 기준값**:

| 발달단계 | 평균 기간 | 범위 | 해석 |
|---------|---------|------|------|
| **초등 저학년** | 3-5일 | 1-7일 | 즉시 소비 |
| **초등 고학년** | 10-15일 | 7-20일 | 계획적 사용 시작 |
| **중학생** | 20-25일 | 15-30일 | 월말까지 계획 |
| **고등학생** | 25-30일+ | 월 전체 | 저축 병행 |

**중요성**:
- 이 지표는 **충동 조절 능력**을 가장 직관적으로 보여줍니다.
- 증가 추세 = 정상 발달
- 정체 또는 악화 = 개입 필요

---

### 3.2 저축 지속 기간 (Savings Duration)

**정의**: 특정 목표를 위한 저축이 얼마나 오래 유지되는가

**측정 방법**:
```
1. 목표 설정 시점 기록
2. 목표 달성 또는 포기할 때까지 추적
3. 지속 기간 계산
```

**발달단계별 기준값**:

| 발달단계 | 달성 가능 기간 | 평균 달성률 |
|---------|--------------|-----------|
| **초등 저학년** | ~1개월 | 30% |
| **초등 고학년** | 1-3개월 | 60% |
| **중학생** | 3-6개월 | 70% |
| **고등학생** | 6-12개월+ | 75% |

**아이부자 추적 예시**:

```
목표: "휴대폰 기기변경 저축" (목표액 500,000원)

설정: 2025-01-15
진행도:
- 1개월 후 (2월): 50,000원 (10%)
- 2개월 후 (3월): 120,000원 (24%)
- 3개월 후 (4월): 200,000원 (40%)

지속 기간: 3개월 ✓
위험도: 낮음 (계속 진행 중)
```

---

### 3.3 거래 빈도 (Transaction Frequency)

**정의**: 월간 얼마나 자주 거래(구매 또는 저축)를 하는가

**측정 방법**:
```
월별 총 거래 건수 = 구매 거래 + 저축 거래 + 기타 거래
```

**발달단계별 기준값**:

| 발달단계 | 월간 거래 | 일일 평균 | 해석 |
|---------|---------|---------|------|
| **초등 저학년** | 25-35회 | 1-1.5회 | 충동적 구매 |
| **초등 고학년** | 15-25회 | 0.5-0.8회 | 계획성 증가 |
| **중학생** | 12-18회 | 0.4-0.6회 | 신중한 선택 |
| **고등학생** | 8-12회 | 0.3-0.4회 | 목적 지향 |

**변화 추이 분석**:
```
거래 빈도 추이:
1월: 32회
2월: 28회 ▼
3월: 22회 ▼
4월: 18회 ▼

해석: "충동 조절 능력이 점진적으로 증가"
```

---

### 3.4 구매 전 대기 기간 (Pre-Purchase Waiting Period)

**정의**: 물건을 "사고 싶다"고 생각한 후 실제 구매까지의 기간

**측정 방법**:

아이부자 앱 내 "관심 목록" 또는 "장바구니" 기능 활용:
```
1. 아이가 관심 상품을 "보류" 표시
2. 자동으로 관심 추가 시간 기록
3. 실제 구매 또는 포기 시점 추적
4. 대기 기간 자동 계산
```

**발달단계별 기준값**:

| 발달단계 | 평균 대기 | 최대 대기 | 해석 |
|---------|---------|---------|------|
| **초등 저학년** | 즉시 | 1-2일 | 거의 충동적 |
| **초등 고학년** | 1-2일 | 5-7일 | 간단한 검토 |
| **중학생** | 3-5일 | 10-14일 | 신중한 재고 |
| **고등학생** | 1주일+ | 3주 이상 | 깊이 있는 검토 |

**아이부자 활용 예시**:

```
관심 목록: "게임 기기" (50,000원)
추가한 날: 2025-03-15
대기 기간: 23일
상태: 아직 구매 안 함

피드백:
"3주를 고민했군요.
정말 필요한지 다시 한 번 생각해보세요.

경우 1) 구매한 경우
→ '충동적 구매를 계획적으로 변경했어요!' ✓

경우 2) 포기한 경우
→ '신중한 판단 능력이 좋네요!' ✓"
```

## 4. 패턴 기반 지표

### 4.1 부모 상담 비율 (Parent Consultation Rate)

**정의**: 전체 거래 중에서 부모의 승인/상담을 거친 비율

**측정 방법**:

아이부자 앱의 "부모에게 물어보기" 기능 기반:
```
부모 상담 비율 = (부모 상담 거래 수 / 전체 거래 수) × 100%
```

**발달단계별 기준값**:

| 발달단계 | 상담 비율 | 상담 대상 금액 | 해석 |
|---------|---------|--------------|------|
| **초등 저학년** | 80-100% | 거의 모든 거래 | 부모 의존적 |
| **초등 고학년** | 50-70% | 10,000원 이상 | 부분적 독립 |
| **중학생** | 20-40% | 20,000원 이상 | 대부분 자율 |
| **고등학생** | <20% | 30,000원 이상 | 거의 완전 자율 |

**아이부자 추적**:

```
이번 달 거래 분석:

총 18회 거래 중:
부모 상담 거래: 6회 (33%)

내역:
1. 간식 (3,000원): 상담 X
2. 게임과금 (5,000원): 상담 O
3. 책 (12,000원): 상담 O
4. 신발 (25,000원): 상담 O
...

평가: "중학생 수준의 독립적 결정 능력"
```

**중요 신호**:
- **급격한 감소**: 충동 구매 위험 증가 신호 (점검 필요)
- **점진적 감소**: 정상적 독립성 증가 신호

---

### 4.2 거래 후회 비율 (Transaction Regret Rate / 반품률)

**정의**: 구매 후 반품, 환불, 또는 "이걸 사지 말았으면" 평가한 비율

**측정 방법**:

아이부자 앱 내 구매 평가 시스템:
```
아이가 각 구매를 "만족" 또는 "후회"로 평가
반품률 = (후회 거래 수 / 전체 거래 수) × 100%
```

**발달단계별 기준값**:

| 발달단계 | 반품률 | 의미 |
|---------|--------|------|
| **초등 저학년** | 20-40% | 충동 구매 많음 |
| **초등 고학년** | 10-20% | 계획성 향상 |
| **중학생** | 5-10% | 신중한 결정 |
| **고등학생** | <5% | 높은 의사결정 능력 |

**아이부자 분석**:

```
이번 달 구매 평가:

총 15회 구매
만족: 12회 (80%) ✓
후회: 3회 (20%) △

후회 항목 분석:
1. "게임 과금" - 충동적 구매
2. "유행 악세서리" - 친구 따라함
3. "다이어트 식품" - 실제로 안 먹음

팁: 이 3가지 카테고리는
48시간 대기 규칙을 적용해보세요!"
```

## 5. 목표 기반 지표

### 5.1 활성 목표 수 (Active Goals)

**정의**: 현재 진행 중인 저축 목표의 개수

**의미**: 장기 계획 능력을 반영

**발달단계별 기준값**:

| 발달단계 | 활성 목표 | 권장 |
|---------|---------|------|
| **초등 저학년** | 0-1개 | 1개 (단기) |
| **초등 고학년** | 1-2개 | 2개 (단기 + 중기) |
| **중학생** | 2-3개 | 3개 (단기 + 중기 + 장기) |
| **고등학생** | 2-3개 | 3개+ (다층화) |

---

### 5.2 목표 달성률 (Goal Achievement Rate)

**정의**: 설정한 목표 중 완수한 목표의 비율

**공식**: (달성한 목표 수 / 전체 목표 수) × 100%

**발달단계별 기준값**:

| 발달단계 | 평균 달성률 | 목표 |
|---------|-----------|------|
| **초등 저학년** | 30% | 40% |
| **초등 고학년** | 60% | 70% |
| **중학생** | 70% | 80% |
| **고등학생** | 75% | 85% |

| **저축률** | 5% | 20% | 35% | 40% |
| **사용률** | 95% | 80% | 65% | 60% |

| **거래 빈도** | 30회/월 | 20회/월 | 15회/월 | 10회/월 |
| **구매 전 대기** | 0일 | 1-2일 | 3-5일 | 1주일+ |
| **부모 상담 비율** | 90% | 60% | 30% | 15% |
| **반품률** | 30% | 15% | 8% | 3% |
| **저축 지속 기간** | <1개월 | 1-3개월 | 3-6개월 | 6-12개월+ |
| **카테고리 다양성** | 1-2개 | 2-3개 | 3-4개 | 4개 이상 |

### 8.2 발달단계 자동 판정 알고리즘

```
입력: 지난 3개월 누적 지표 데이터

단계 1) 초등 저학년 vs 고학년 판정
IF (저축률 < 10% AND 용돈소진기간 < 7일) THEN
   return "초등 저학년"

단계 2) 초등 고학년 vs 중학생 판정
IF (저축률 >= 20% AND 저축률 < 35%
    AND 부모상담비율 >= 50%) THEN
   return "초등 고학년"

단계 3) 중학생 vs 고등학생 판정
IF (저축률 >= 35% AND 부모상담비율 <= 40%
    AND 반품률 <= 10%) THEN
   return "중학생"

단계 4) 고등학생 판정
IF (저축률 >= 40% AND 부모상담비율 <= 20%
    AND 거래빈도 <= 12회) THEN
   return "고등학생"

출력: 발달단계 + 확신도(%) + 권장사항
```
