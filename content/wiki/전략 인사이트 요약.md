## 아이부자의 핵심 포지셔닝

> [!quote] 전략 핵심 명제
> "아이부자는 아이의 성장과 함께 진화하는 금융 서비스다.
> 영유아 → 초등 저학년 → 초등 고학년 → 중학생 → 고등학생
> 각 단계마다 기능이 확장되고, 교육이 심화되고, 부모-자녀 상호작용이 변화한다.
> 이것이 경쟁사가 따라올 수 없는 근본적인 차별화다."

## 3대 차별화 전략

### 차별화 ①: 법정 기준 기반 체계적 교육

**핵심 메시지**: "아이부자의 교육 콘텐츠는 금융감독원 표준과 2022 개정 교육과정 기반"

| 구분 | WON | 헤이영 | 리브Next | **아이부자** |
|------|:---:|:------:|:--------:|:-----------:|
| 초등 기초 교육 | ✗ | △ | ✗ | ✅ 체계적 |
| 중학 시뮬레이션 | ✗ | ✗ | ✗ | ✅ |
| 고등 투자 교육 | ✗ | △ | ✓ | ✅ 통합 |
| 부모 교육 가이드 | ✗ | △ | ✗ | ✅ 발달단계별 |
| 법정 기준 연계 | ✗ | ✗ | ✗ | ✅ 공식 인증 |

**근거 논문**:
- 박형준(2022): 한국과 미국 금융교육 표준안 비교 → 한국 표준의 체계화 가능성
- 홍현의&최정민(2021): 시뮬레이션 4단계 학습 모드가 추상적 지식을 경험으로 전환

### 차별화 ②: 발달단계별 자동 상호작용 패러다임

**핵심 메시지**: "시스템이 자녀 행동 데이터를 분석해 부모에게 '언제, 어떻게 지도할지' 자동 제시"

**측정 지표 (v2.0)**:
- 월간 저축률 추이
- 용돈 소진 기간
- 거래 빈도·패턴
- 목표 달성률

단계 자동 전환 기준 → [[서비스측정가능지표정리]] 참고

### 차별화 ③: 미래 고객 파이프라인

**하나은행의 장기 전략 자산**:

```
청소년 사용자 (아이부자)
       ↓ 18세 이후
하나은행 성인 금융 고객
       ↓
평생 고객 관계
```

- RBC Mydoh: 6~14세 → 성인 계좌 전환 전략으로 성공
- JPMorgan Chase: 자녀 계좌 + 부모 계좌 필수 연동

## 교육 우선순위 매트릭스

청소년의 금융행태 합리성 수준별 교육 우선순위 (정영선, 2012):

| 영역 | 합리성 수준 | 우선순위 | 아이부자 콘텐츠 방향 |
|------|:----------:|:--------:|------------------|
| **예산관리** | 낮음 (50점 미만) | 🔴 최우선 | 월간 용돈 계획표, 지출 카테고리 분석, 충동구매 경보 |
| **신용관리** | 낮음 (50점 미만) | 🟡 우선 | 신용카드 vs 체크카드 차이, 신용점수 개념 |
| **저축행태** | 높음 | 🟢 유지·강화 | 기존 저축 목표 기능 고도화 |

## 청소년 투자자 유형별 서비스 설계

정유진(2024)의 4가지 유형:

| 유형 | 핵심 니즈 | 아이부자 설계 방향 |
|------|---------|-----------------|
| **공격형** | 빠른 거래, 고수익 | 위험관리 교육 의무화, 손실 한도 설정 |
| **손실회피형** | 안정성, 손실 공포 | 장기투자 시뮬레이션, 분산투자 안내 |
| **가치투자자** | 기업 분석, 정보 | 재무정보 요약 카드, 포트폴리오 분석 |
| **모멘텀투자자** | 트렌드 추종 | 비판적 정보 평가 교육, 독립 판단 훈련 |

## 실행 로드맵

### Phase 1 (현재~6개월): 기초 교육 기반 구축

```
✅ v2.0 시스템 측정 지표 완성
✅ 초등 저학년 교육 콘텐츠 5개 제작
✅ 부모 가이드 문서 3개 (단계별)
✅ 자동 알림 시스템 구현 (기초)

목표:
- 교육 콘텐츠 완수율 40%+
- 부모 만족도 4.0/5.0+
```

### Phase 2 (6~12개월): 단계별 완전화

```
✅ 초등 고학년 목표 저축 기능 고도화
✅ 중학생 시뮬레이션 기능 추가
✅ 부모 자동 피드백 로직 고도화
✅ 법정 기준 공식 인증 획득

목표:
- 목표 달성률 60%+
- 발달단계 자동 인식 정확도 80%+
```

### Phase 3 (12~18개월): 투자 플랫폼 통합

```
✅ 고등학생 실제 투자 기능 구현
✅ 모의투자 → 실제투자 전환 로직
✅ 교육부 "금융과 경제생활" 교과 연계
✅ 커뮤니티 투자 정보 플랫폼

목표:
- 고등학생 세그먼트 MAU 50만+
- 투자 상품 사용률 30%+
```
