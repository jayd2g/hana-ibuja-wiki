## 타겟 사용자 구조

```
아이부자 사용자 생태계

┌─────────────────────────────────────────┐
│  자녀 (직접 사용자)                      │
│  ├── 영유아 (0~5세)   → 부모 대리 사용  │
│  ├── 초등 저학년 (6~8세) → 주요 시작점  │
│  ├── 초등 고학년 (9~12세) → 핵심 사용층│
│  ├── 중학생 (13~15세) → 의사결정 주체화│
│  └── 고등학생 (16~18세) → 투자 진입    │
│                                         │
│  부모 (간접 사용자 + 파트너)             │
│  └── 모니터링, 교육 지도, 공동 사용     │
└─────────────────────────────────────────┘
```

## 발달단계별 특성

### Stage 1: 영유아 (0~5세)

| 항목 | 특성 |
|------|------|
| 인지 발달 | Piaget 전조작기 — 추상 개념 이해 불가 |
| 금융 인식 | 금융 개념 전무 |
| 부모 역할 | 절대적 영향력 (초기 금융태도 형성의 임계기) |
| 핵심 과제 | 돈의 존재 인식, 교환 개념 시작 |

### Stage 2: 초등 저학년 (6~8세) ⭐ 황금 기간

| 항목 | 특성 |
|------|------|
| 인지 발달 | Piaget 구체적 조작기 — 구체적 거래 이해 가능 |
| 금융 행동 | 용돈 수령, 소액 구매 시작 |
| 부모 역할 | 가장 효과적인 교육 시기 (또래 압박 약함) |
| 핵심 과제 | 저축통 활동, 목표 설정, 기다림 연습 |

> [!tip] 황금 기간
> 이 시기부터 독립적 금융 거래 시작 가능하고, 실패해도 영향이 작아 안전한 학습 가능

### Stage 3: 초등 고학년 (9~12세)

| 항목 | 특성 |
|------|------|
| 인지 발달 | 추상적 사고 발달 시작 |
| 금융 행동 | 가계부, 목표 저축, 카테고리별 지출 관리 |
| 또래 영향 | 부모(3.08점) ≈ 또래집단(2.81점) 영향 수렴 |
| 핵심 과제 | 예산 계획, 충동 구매 조절 |

### Stage 4: 중학생 (13~15세)

| 항목 | 특성 |
|------|------|
| 인지 발달 | 형식적 조작기 — 추상적·미래 계획 가능 |
| 금융 행동 | 신용 개념, 의사결정 시뮬레이션, 모의투자 |
| 자율성 욕구 | 자기결정 강화 — 부모의 설명 방식 변화 필요 |
| 핵심 과제 | 금융 사기 예방, 신용카드 이해 |

### Stage 5: 고등학생 (16~18세)

| 항목 | 특성 |
|------|------|
| 금융 행동 | 실제 투자 진입, 독립적 포트폴리오 |
| 투자 유형 | 공격형 / 손실회피형 / 가치투자자 / 모멘텀투자자 |
| 핵심 과제 | 자기주도 의사결정, 성과 피드백 |

## 청소년 금융이해력 잠재집단 (박상은, 2020, n=707)

```
집단 3 (50.2%) ████████████████████  낮은 지식·태도 + 높은 행동
집단 2 (32.7%) █████████████          높은 금융이해력
집단 1 (17.1%) ███████                낮은 금융이해력
```

> [!warning] 시사점
> 청소년의 절반(50.2%)은 **이해 없이 행동**하는 상태 → 행동에 교육 안내(scaffolding)를 결합한 설계가 필수

## 금융태도 4가지 유형 (장연주·김민정, 2023)

| 유형 | 책임감 | 걱정 | 비율 | 특징 |
|------|:------:|:----:|:----:|------|
| **성취지향형** | 높음 | 높음 | ~25% | 목표설정·자기관리 우수 |
| **취약형** | 낮음 | 높음 | ~20% | 불안감, 신중성 강조 필요 |
| **자기과신형** | 낮음 | 낮음 | ~30% | 위험관리 교육 필요 |
| **안정형** | 높음 | 낮음 | ~25% | 균형잡힌 접근 |

Z세대 → "성취지향형" 비율 높음 (타 세대 대비)

## 금융태도에 영향을 미치는 요인

### 긍정적 요인 (강도 순)

1. **부모 영향** ← 가장 강력
   - 부모의 금융태도 → 자녀 모델링
   - 가정 내 금융 대화 빈도 ↑ → 금융태도 ↑
2. **실제 거래 경험** — 본인 명의 통장 보유 시 책임감 증가
3. **금융교육** (체험형) — 시뮬레이션·실습 연계 방식
4. **또래집단** — 초등 고학년 이후 영향력 증가

### 유의 사항

- 단순 지식 전달형 학교 교육은 **금융행태에 유의미한 효과 없음** (정영선, 2012)
- 학교·미디어를 통한 금융사회화는 태도에 **부적(-)** 영향 가능 (조선영·최현자, 2023)

## 디지털 환경의 추가 요구 역량 (조선영, 2025)

| 역량 | 내용 | 아이부자 구현 방향 |
|------|------|-----------------|
| 디지털 금융거래 역량 | 앱 기반 재무자산 관리 | 직관적 UI, 거래 이력 대시보드 |
| 정보 비판적 평가 역량 | 금융 정보 검증·판단 | 팩트체크 콘텐츠, 출처 표시 |
| 디지털 금융위험 대응 역량 | 사기·피해 인식 및 대응 | 이상거래 탐지, 사기 예방 교육 |

## 금융소비자경험 3단계 (조선영, 2025)

```
① 금융시장 유입기   →  ② 금융생활 탐험기   →  ③ 금융생활 독립준비기
   (준비 없이 진입)       (다양한 역할 수행)       (자기주도 의사결정)

아이부자 역할:
① 안전한 첫 경험    →  ② 경험 다양화 지원  →  ③ 독립 자신감 강화
```
