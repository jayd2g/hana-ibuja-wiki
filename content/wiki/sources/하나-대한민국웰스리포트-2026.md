## 핵심 요약

하나은행 하나금융연구소 「대한민국 웰스 리포트」 **18호(2026.4 발간)**. 부자 713명, 대중부유층 1,355명, 일반대중 645명 — **총 2,713명** 온라인 설문 + 하나은행 PB·세무사·손님 인터뷰 25명. 3부 구성: **FOCUS — K-EMILLI**(최근 10년 내 부자 반열), **INSIGHT — 2026 전망·PB·AI·2025 회고**, **SPECIAL — 상속·증여 + 소셜 자본(부자의 커뮤니티)**.

**★ 아이부자 전략에 결정적 가치 2개**:
1. **K-EMILLI 프로필** = "회사원+공무원 30%, 30평형 이하 아파트 44%, 평균 자산 60억" → **아이부자 중산위층 타겟(연소득 8천만~2억)과 거의 일치하는 "근접 부자"층**이 실재하고 확대 중임을 증명.
2. **상속·증여 시점 3위: "어릴 때부터 수시로" 31%** + 60대 여 인터뷰 **"자녀 어릴 때부터 10년 주기 증여, 결혼할 때 1억원 지급"** — **미성년 10년 2천만원 증여세 면제 한도**를 실제로 사용하는 한국 부자 가정의 구조적 실행 근거. 2025 하나 리포트(영리치 미성년 주식 5%)의 **"결과(행동)"**와 짝을 이루는 **"실행 구조(증여 타이밍)"** 데이터.

## Part 1 — FOCUS: K-EMILLI (Korea Everywhere Millionaires)

### K-EMILLI 정의
- **기준**: 최근 **10년 내** 부자 반열에 올라선 만 50세 이하
- **포함 범위**: 금융자산 10억 원 이상 + 총자산 30억/금융자산 5억 이상 = 총 **243명** 분석
- 개념적으로 "동네 어디에나 있을 법한 보통의 젊은 부자" (Korea Everywhere Millionaires)

### K-EMILLI 프로필
| 항목 | 수치 |
|------|------|
| 평균 연령 | 51세 |
| 남성 | 64% |
| 대학원 졸업 | 41% |
| 연평균 소득 | **5억 원** |
| 총자산 | 60억 원 대 |
| 거주 형태 | **30평형 이하 아파트 44%** (수도권 집중) |
| 직업 구성 | **회사원·공무원 30%**, 자영업·사업가 42%, 전문직 21% |

**★ 결정적 함의**: "회사원+공무원 30%"와 "30평형 이하 아파트 44%" — K-EMILLI는 **재산이 많아 보이는 사람들이 아니라, 꾸준한 소득과 저축·투자로 10년 안에 도달한 근접층**. 아이부자 타겟인 "상위 중산층·중산위층 부모"가 실제로 **10년 뒤 K-EMILLI로 진입할 수 있는 경로가 실증**됨.

### 종잣돈·부의 기반
- **종잣돈 평균 8.5억 원** (현재 총자산의 약 10%)
- **종잣돈 마련 방법 TOP 5**: ①저축 43% ②소득인상 19% ③상속·증여 19% ④부동산 10% ⑤주식 8%
- **부의 기반**:
  - 상속자산 위주 **28%**
  - 상속 있으나 노력이 주 **22%**
  - 내 노력 위주 **50%**

### 부 형성 기여 요인 TOP 5
1. 소득 인상(사업 호황·연봉) 44%
2. **주식 투자 성공 36%** (⬅ 핵심)
3. 저축 28%
4. 거주주택 가격 상승 24%
5. 상속·증여 21%
6. 부동산 투자 11%
7. 현물자산 상승 6%
8. 스타트업 투자 3%

### "평범한 사람이 부자 되기 어려워졌다" 동의율
- K-EMILLI **66%** (일반 부자 60%)
- **5년 내 부자는 66%가 가장 강하게 동의** → 최근 진입자가 "이 경로가 점점 닫히고 있다"는 위기감을 가장 크게 느낌 → **지금 자녀에게 조기 시작 기회를 주는 것이 결정적이라는 인식의 근거**

### K-EMILLI의 기부 실천
- 정기 기부 실천 **10년 내 부자 62%** vs 10년 이상 부자 58%
- K-EMILLI 평균 기부금 **연 1.2%** (일반 부자 0.7%의 **1.8배**)
- → 부 형성 과정에서 "사회적 책임" 내재화 수준이 더 높음

## Part 3 — SPECIAL ①: 상속·증여

### 자산이전 계획
- 자산이전 계획 있음 **82%**
- 후대 이전 **68%** (증여+상속 57% + 전부상속 7% + 전부증여 4%)
- 사회 환원 11%, 전부 나 위해 3%

### "재산 많이 물려줄수록 성장 기회 생긴다" 동의율
- 전체 **68%**
- 연령대별: 40대↓ **71%**, 50대 70%, 60대 65%, 70대 66%
- **상속형 부자 73%** vs 자수성가형 63%
- → **젊은 세대일수록 조기 이전이 기회라는 인식 강함**

### 증여 시점 TOP 4 — ★★★
| 순위 | 증여 시점 | 비율 |
|------|-----------|------|
| 1 | 물려줄 대상 주택 구입·사업 목돈 필요 시 | 41% |
| 2 | 결혼 시 | 33% |
| **3** | **어릴 때부터 수시로** | **31%** |
| 4 | 정책상 절세 유리할 때 | 29% |

**→ "시점 분산 여러 번 증여" 동의 57%**

**★ 이 한 줄이 아이부자 전략의 결정적 근거**: 한국 부자 가정 **31%가 이미 "어릴 때부터 수시로" 증여를 실행 중**이라는 실증 데이터. 해외 HNWI 연구(Wells Fargo·Fidelity)의 "조기 자산 이전 구조"가 한국에서도 동일하게 관측됨.

### 상속·증여 자산 유형 (복수응답)
| 유형 | 비율 |
|------|------|
| 현금·예금 | **80%** |
| 부동산(거주용) | 39% |
| 부동산(거주외) | 31% |
| **주식** | **29%** |
| 보험·연금 | 22% |
| 금·귀금속 | 20% |
| 채권·펀드 | 12% |
| 사업체 | 6% |
| 가상자산 | 3% |

**→ 현금·예금 80% + 주식 29% = 자녀 명의로 즉시 운용 가능한 자산이 이전의 주력**. 부동산이 아니라 **자녀가 직접 관리·학습할 수 있는 자산 형태**.

**50대 이하 부자는 주식으로 이전 의향이 더 높음** — 세대 교체와 함께 주식 이전이 주류화되는 중.

### 60대 여 부자 실제 인터뷰 (★ 미성년 증여 한도 실행 사례)
> "자녀 어릴 때부터 10년 주기로 증여하며, 결혼할 때 1억원 지급. 절세 범위 내에서 계속 증여."

- 한국 세법 상 **미성년 자녀 증여세 면제 한도 10년간 2천만원** 구조를 정확히 활용
- **자녀 나이 0~10세 2천만원 → 10~20세 2천만원 → 20~30세 5천만원 → 결혼 시 1억원 추가 공제** = 결혼 전까지 누적 비과세 증여 약 **1.4억~1.5억**
- → 한국 부자 가정이 이미 **"만 0세부터 증여"**를 실행 중임을 보여주는 실증 인터뷰

### 상속 우려 사항 (복수응답)
| 우려 | 비율 |
|------|------|
| 상속세·증여세 부담 | **79%** |
| 정책·법률 제도의 잦은 변화, 모호한 기준 | 52% |
| 상속·증여 절차의 복잡성 | 46% |
| 상속할 자산의 미래 가치 예측 어려움 | 27% |
| 상속·증여 자산의 관리·운용 자신감 부족 | **19%** |
| 가족 간 분란·법적 분쟁 우려 | 18% |
| 상속세 납부 재원 부족 | 17% |

**→ 세무사·PB 자문 수요 증가**. "상속·증여 자산의 관리·운용 자신감 부족 19%"는 자녀가 **증여받은 후 운용할 수 있는 역량**이 필요함을 시사 → **아이부자가 "증여 이후의 자녀 운용 학습 도구"로 포지셔닝할 여지**.

### 최호준 아트레온 회장 인터뷰 (상속의 새 정의)
> "부친의 물질적 자산을 받기 전에 **정신적 자산**을 먼저 물려받았다. 그리고 받은 자산을 나누는 데 힘을 기울이고 싶다."

- 부자의 진정한 대물림은 재산이 아닌 **삶의 가치관·태도**
- **헤리티지(Heritage)** 개념으로 상속을 재정의
- → 아이부자 전략의 "The Wealth Apprentice" 내러티브와 직접 연결됨. "돈이 아니라 돈을 다루는 태도·경험을 물려준다"는 메시지 근거.

## Part 3 — SPECIAL ②: 소셜 자본 · 부자의 커뮤니티

### 정기 모임 참여
- 부자 **83%** vs 일반대중 66% (부자가 1.26배)
- 부자의 1/3은 1~2개, 1/3은 3~4개, 나머지는 5개 이상 활발히 참여
- **자산·소득 규모 비례 상승**: 10억↓ 63% → 100억+ **87%** / 5천만원↓ 72% → 2억+ **84%**
- **연령별**: 30대↓ 57%, 40대 62%, 50대 81%, 60대 89%, 70대+ **93%**
- 지역: 서울 86% > 수도권 82% > 지방광역시 74%

### 모임 유형 (부자 우위)
| 유형 | 부자 | 일반대중 | 격차 |
|------|------|----------|------|
| 동문·동창 모임 | **59%** | 42% | +17%p |
| 가족·친척 모임 | 38% | 32% | +8%p |
| 취미·동호회 | 37% | 30% | +7%p |
| **비즈니스 교류** | **20%** | 8% | **+12%p** |
| 친목 모임 | 46% | 51% | -5%p (대중 우위) |

**→ 부자의 모임은 "폐쇄적 네트워크"(동문·비즈니스) 중심** / 참여 자격 제한된 모임: 총자산 규모별 42~49%.

### 모임 비용
- 부자: 월 평균 **3회, 56만 원** (대면 모임 중심)
- 일반대중: 월 평균 2회, 18만 원 (부자의 1/3 수준)
- 자산 100억+ 월 **87만 원**, 70대+ 월 **83만 원**

### 모임의 주된 가치
| 가치 | 비율 |
|------|------|

### 아이부자 관련 함의 — "모임 = 가족 내 자산 관리 사이클의 원형"
- 부자 가정은 **"가족·친척 모임" 38%**로 일반대중 32%보다 높음 → 가족이 자산·가치관을 공유하는 **장치**로 기능
- 부자의 월 56만원·3회 대면 모임 = **"가족 금융 회의"**가 확대된 구조
- 아이부자 앱 내 **"가족 자산 공유 피드·가족 기념일별 증여 알림·가족 미션"** 기능이 부자 가정의 소셜 자본 구조를 **디지털 모임 형태로 이식**하는 도구가 될 수 있음 → Page 22 Element 03 "가족·자산 네트워크"의 직접 근거 보강

### Family Office Leaders Community (하나은행 자체 프로그램)

### ★ Page 20 3줄 (부모·아이·하나은행) 직접 근거

### ★ Page 18 제약(일반 가정의 인식 격차) — 근거 보강
- 한국 부자의 상속 우려 1위는 **"세부담 79%"** → 일반 가정은 "상속·증여는 큰 자산가 일" 인식에 머물러 있으나, **미성년 10년 2천만원 비과세 구조**는 누구에게나 열려 있음
- "상속·증여 자산 관리·운용 자신감 부족 19%"는 **부자 가정조차 자녀 운용 역량에 불안**을 느낀다는 의미 → 일반 가정은 더 심각 → **아이부자가 이 격차를 메우는 교육·운용 도구** 포지셔닝 가능

### ★ Element 구조와의 매핑
- **Element 00 (진입·기본 UX)**: K-EMILLI의 先공부後투자 47% → "먼저 배우고 시작한다"는 행동 원칙이 앱 온보딩 UX 근거
- **Element 01 (최초의 자산 관리 사이클)**: K-EMILLI의 종잣돈 8.5억 평균 / 부 형성 2위 요인 "주식 투자 성공 36%" → **"주식 투자 성공 경험"이 실제 부 형성의 핵심 트리거**
- **Element 02 (성장 및 확대)**: K-EMILLI의 자산 확대 노력 TOP — 절세 42%, 금융투자 공부 41%, 리밸런싱 38% → 앱 기능 로드맵의 구성 요소
- **Element 03 (가족·자산 네트워크)**: K-EMILLI **"가족 단위 자산관리 21%"** + 부자 "가족·친척 모임 38%" + 상속·증여 자산이전 계획 82% → 가족 단위 자산 네트워크가 **한국 부자의 실제 운용 방식**

### ★ 트랙 A (구조적 자산이전) — 강력한 근거
- **미성년 증여 10년 2천만원 비과세 한도**가 60대 여 부자 인터뷰에서 실제 사용 중
- **"어릴 때부터 수시로" 증여 31%**는 트랙 A의 한국 내 실재 수요를 입증
- 아이부자가 "증여 등록 → 증여세 신고 → 자녀 명의 운용" 구조를 제공하면 상위 중산층도 부자 가정의 구조를 그대로 이식 가능

### ★ 핵심 미해결 질문 "부모가 앱을 계속 여는 이유"에 대한 단서
- 부자 **수익 재투자 94%** vs 대중부유층 86% vs 일반대중 74% → **재투자 습관의 차이가 부의 격차를 만든다**
- 부자의 수익 용처 중 "자녀·부모님 용돈 15%" → 투자 수익이 **가족 내 순환**하는 구조
- → **부모가 앱을 여는 이유 = "내 투자 수익이 자녀 자산으로 연결되는 순환"을 매일 확인하고 싶어서** — 이 메커니즘을 앱 내에서 시각화하면 개방 동기 형성 가능

**★ 3년 연속 주제 추이의 의미**: 하나금융연구소는 **"부자가 누구이며, 어떻게 만들어지는가"**의 초점이 **"내면·삶" → "세대 차이" → "진입 경로"**로 이동 중. 이는 **아이부자 전략이 겨냥하는 "중산위층도 부자가 될 수 있는 조기 학습 구조"** 담론과 정확히 동기화.

**18호의 상속·증여 섹션**은 17호의 영리치 섹션과 직접 이어짐:
- 17호: 영리치는 **미성년 시절부터 주식 운용** (결과 데이터)
- 18호: 부자 가정은 **"어릴 때부터 수시로" 증여** (원인·실행 구조 데이터)
- **→ 결합 논리**: 부자 가정이 어릴 때부터 증여 → 자녀는 학창시절부터 주식 운용 → K-EMILLI 또는 영리치로 진입

이 3개년 데이터 체인이 아이부자 전략의 "조기 시작"·"가족 단위 운용"·"증여 구조"·"주식 경험"을 **한국 내 실증 데이터로 완결**시킴.
